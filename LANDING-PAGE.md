| 항목 | 내용 |

## 디자인 시스템

### 컬러

| 용도 | HEX | 설명 |
|------|-----|------|
| 배경 (메인) | `#0a0e17` | 쌤핀 앱과 동일한 딥 네이비 |
| 배경 (섹션) | `#0f1729` | 약간 밝은 네이비 (교차 섹션) |
| 카드 배경 | `#1a2332` | 기능 카드 |
| 강조 (CTA) | `#3b82f6` | 다운로드 버튼 (파란색) |
| 강조 호버 | `#2563eb` | 버튼 호버 |
| 서브 강조 | `#f59e0b` | 앰버/골드 (하이라이트, 배지) |
| 텍스트 | `#f8fafc` | 메인 텍스트 (밝은 흰색) |
| 서브 텍스트 | `#94a3b8` | 설명 텍스트 (슬레이트) |
| 그라데이션 | `#3b82f6 → #8b5cf6` | 히어로 배경 그라데이션 악센트 |

### 폰트

- **한글**: Pretendard 또는 Noto Sans KR
- **제목**: Bold / ExtraBold
- **본문**: Regular / Medium
- CDN: `<link>` 또는 `next/font`

### 공통 스타일

- 최대 너비: `max-w-6xl` (1152px) + `mx-auto`
- 섹션 간격: `py-20` ~ `py-24`
- 카드 모서리: `rounded-2xl`
- 그림자: 다크 테마이므로 미묘한 border (`border border-white/10`) 위주
- 애니메이션: 스크롤 시 fade-in-up (Intersection Observer 또는 framer-motion)
