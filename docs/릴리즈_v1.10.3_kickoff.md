## 📦 릴리즈 범위

### 포함 (2종 모두 **베타**)

1. **협업 보드 (collab-board)** — Phase 1a MVP 전체
   - Jamboard 대안. 학생과 실시간 드로잉 협업
   - UI에 `BETA` 배지 + 안내 배너 이미 적용 완료 (commit `7fb372c`)
   - 상세: [HANDOFF_collab-board.md](./HANDOFF_collab-board.md), [collab-board.report.md](./04-report/features/collab-board.report.md)

2. **복합 유형 설문 대폭 개선 (multi-survey Phase 1+2)** — 실제 산출물 4가지:
   - **스프레드시트 뷰** — 결과를 요약/테이블/개별 3탭으로 분석 (정렬·필터·검색·키보드 내비 ←/→)
   - **Excel 내보내기** — 요약/전체응답/워드빈도 3시트 구성, 파일명 자동 sanitize
   - **피드백 프리셋 3종** — "발제 퀵 피드백" / "사례나눔 리플렉션" / "소풍 활동 소감"
   - **라이브 피드백 월 (`FeedbackWallView`)** — 프로젝터용 포스트잇 월, 텍스트 응답 실시간 카드 + 하이라이트/고정/장문확대 모달
   - 구 ResultsView 약 480줄 제거, `PastResultsView`에 "스프레드시트로 열기" 모달 진입점 추가
   - Phase 3(이미지 응답·AI 그룹핑·Sheets 연동 등)는 다음 버전 이월 ([발제피드백_Phase3_kickoff.md](./발제피드백_Phase3_kickoff.md))
   - **UI 베타 표시 아직 미적용 — 릴리즈 세션에서 추가 필요 (§2 아래 참조)**

3. **Vitest 테스트 인프라 (chore)** — 개발 품질 기반
   - `vitest.config.ts` + test/test:watch/test:ui 스크립트
   - 발제피드백 도메인 규칙·Excel 직렬화에 **52개 단위 테스트** (aggregation 22 + serialization 19 + exporter 11) 추가
   - 앞으로 기능 추가 시 회귀 방지 기반 확보

### 제외

- 발제피드백 Phase 3 (다음 버전)
- `feature/mobile`, `feature/todo-pro-mode` 브랜치 — **이미 main(v1.10.2)에 머지되어 있음** (ancestor 확인 완료). 로컬 브랜치 포인터만 남음. 별도 동반 필요 없음. 릴리즈 후 `git branch -d`로 정리 가능.
