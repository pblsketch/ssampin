## 설치 관련 문제

### Windows Defender SmartScreen 경고

설치 파일을 실행했을 때 "Microsoft Windows의 PC 보호"라는 파란 경고창이 뜨는 경우가 있어요. 이는 쌤핀이 개인 개발 앱이라 Microsoft 공식 인증서가 아직 없기 때문이에요. 악성 소프트웨어가 아니니 안심하고 아래 방법으로 진행하세요.

**해결 방법:**
1. 경고창에서 **"추가 정보"** 텍스트 링크를 클릭하세요.
2. 하단에 나타나는 **"실행"** 버튼을 클릭하세요.
3. 이후 정상적으로 설치가 진행돼요.

> 쌤핀은 오픈소스 프로젝트로, 소스 코드를 GitHub에서 직접 확인할 수 있어요. 안심하고 사용하셔도 돼요!

---

### Windows 11 스마트 앱 컨트롤 차단

Windows 11의 스마트 앱 컨트롤 기능이 설치를 차단하는 경우예요. 스마트 앱 컨트롤은 Microsoft Store 외 앱을 엄격하게 검사해요.

**해결 방법:**
1. 다운로드한 설치 파일(`ssampin-Setup.exe`)을 **우클릭**하세요.
2. **"속성"** 을 선택하세요.
3. 속성 창 하단의 **"차단 해제"** 체크박스에 체크하세요.
4. **"확인"** 을 클릭하세요.
5. 이제 설치 파일을 다시 실행하면 정상적으로 설치돼요.

---

### V3 백신(안랩) 차단

**이 문제는 매우 자주 발생해요!** 안랩 V3 백신이 설치 파일을 **조용히 삭제하거나 차단**하는 경우예요. 아무 경고 없이 파일이 사라지거나 실행이 안 되는 경우 V3가 원인일 가능성이 높아요.

**해결 방법:**
1. 작업 표시줄 우측 하단의 V3 트레이 아이콘을 **우클릭**하세요.
2. **"실시간 검사 일시 중지"** 를 선택하세요.
3. 잠시 중지할 시간을 선택하고 확인하세요.
4. 쌤핀 설치를 진행하세요.
5. 설치가 완료된 후 V3 실시간 검사를 **다시 켜세요**.

**추가 팁:** 설치 완료 후 V3 설정에서 쌤핀을 신뢰할 수 있는 프로그램으로 예외 등록하면 이후에도 차단되지 않아요.

> V3 백신이 설치 파일을 삭제한 경우, 쌤핀 공식 웹사이트(ssampin.vercel.app)에서 다시 다운로드하세요.

---

### 알약(ESTsoft) 차단

알약 백신이 설치 파일을 차단하는 경우예요. V3와 마찬가지로 개인 개발 앱이라 오탐이 발생할 수 있어요.

**해결 방법:**
1. 작업 표시줄 우측 하단의 알약 트레이 아이콘을 **우클릭**하세요.
2. **"실시간 감시 해제"** 를 선택하세요.
3. 쌤핀 설치를 진행하세요.
4. 설치가 완료된 후 알약 실시간 감시를 **다시 켜세요**.

**추가 팁:** 설치 후 알약 설정에서 쌤핀 실행 파일을 검사 예외 목록에 추가하면 앞으로 차단되지 않아요.

---

### 설치 파일을 실행해도 아무 반응이 없음

설치 파일을 더블클릭했는데 아무 일도 일어나지 않는 경우예요.

**원인 및 해결 방법:**

1. **백신이 조용히 차단한 경우 (가장 흔한 원인)**
   - V3, 알약 등 설치된 백신의 실시간 감시를 임시로 해제하세요.
   - 해제 후 설치 파일을 다시 실행하세요.
   - 위의 V3 또는 알약 해결 방법을 참고하세요.

2. **관리자 권한이 필요한 경우**
   - 설치 파일을 **우클릭**하세요.
   - **"관리자 권한으로 실행"** 을 선택하세요.

3. **설치 파일 경로에 한글이 포함된 경우**
   - 예: `C:\사용자\홍길동\다운로드\ssampin-Setup.exe`처럼 한글 경로에 있으면 실행이 안 될 수 있어요.
   - 설치 파일을 `C:\temp\` 같은 영문 경로 폴더로 이동한 후 실행하세요.

---

### 설치 도중 오류 발생

설치 중간에 오류 메시지가 표시되거나 설치가 중단되는 경우예요.

**해결 방법:**
1. **기존 쌤핀이 설치되어 있다면 먼저 삭제하세요.**
   - Windows 설정 → 앱 → 쌤핀 찾기 → 제거
   - 데이터는 별도 폴더(`%APPDATA%\ssampin\data\`)에 보관되므로 삭제해도 데이터는 유지돼요.
2. 설치 파일을 **관리자 권한으로 실행**하세요.
3. 백신이 설치를 방해하고 있다면 실시간 감시를 임시로 해제하세요.
4. 위 방법으로도 안 된다면 컴퓨터를 재시작한 후 다시 시도해 보세요.

---

## 업데이트 관련 문제

### 자동 업데이트가 안 됨

쌤핀은 새 버전이 출시되면 앱 실행 시 자동으로 업데이트 알림이 표시돼요. 자동 업데이트가 동작하지 않는 경우예요.

**확인 사항:**
- 인터넷 연결이 정상인지 확인하세요.
- 회사나 학교 네트워크에서 방화벽이나 프록시가 GitHub 연결을 차단할 수 있어요.

**수동 업데이트 방법:**
1. [ssampin.vercel.app](https://ssampin.vercel.app)에 접속하세요.
2. 최신 설치 파일(`ssampin-Setup.exe`)을 다운로드하세요.
3. 기존 앱을 종료하고 설치 파일을 실행하세요.
4. **데이터는 자동으로 보존돼요.** 새로 설치해도 기존 데이터가 그대로 유지돼요.

---

### 업데이트 후 앱이 시작되지 않음

업데이트 후 앱이 실행되지 않거나 오류가 발생하는 경우예요.

**해결 방법:**
1. **작업 관리자**를 열어주세요 (Ctrl+Shift+Esc).
2. 프로세스 목록에서 **"ssampin"** 관련 프로세스를 찾아 **"작업 끝내기"** 를 클릭하세요.
3. 다시 쌤핀을 실행해 보세요.
4. 그래도 안 된다면 **[ssampin.vercel.app](https://ssampin.vercel.app)에서 최신 버전을 다시 설치**하세요.
   - 기존 데이터는 `%APPDATA%\ssampin\data\` 폴더에 별도 보관되므로 유실되지 않아요.

---

### 수동으로 업데이트하는 방법

자동 업데이트를 기다리지 않고 직접 업데이트하고 싶은 경우예요.

**순서:**
1. [ssampin.vercel.app](https://ssampin.vercel.app)에서 최신 설치 파일(`ssampin-Setup.exe`)을 다운로드하세요.
2. 현재 실행 중인 쌤핀 앱을 종료하세요.
3. 다운로드한 설치 파일을 실행하세요.
4. 설치가 완료되면 자동으로 최신 버전으로 업데이트돼요.

> 기존 데이터는 `%APPDATA%\ssampin\data\` 폴더에 별도로 보관되므로, 업데이트나 재설치를 해도 시간표, 학생 정보, 메모 등 모든 데이터가 그대로 유지돼요.

---

### 업데이트 후 데이터가 사라진 것 같음

업데이트 후 기존에 입력했던 데이터가 보이지 않는 경우예요.

**확인 방법:**
1. Windows 탐색기를 열고 주소창에 `%APPDATA%\ssampin\data` 를 입력하고 Enter를 누르세요.
2. 해당 폴더에 JSON 파일들이 있는지 확인하세요.
3. 파일이 있다면 데이터는 정상적으로 보관되어 있어요. 앱을 재시작하면 데이터가 다시 표시돼요.

**파일이 없는 경우:**
- 앱을 완전히 종료했다가 다시 실행해 보세요.
- 그래도 안 된다면 AI 도우미의 에스컬레이션 기능으로 개발자에게 문의해 주세요.

---

## NEIS 연동 문제

### 시간표가 표시되지 않음

NEIS(교육행정정보시스템)와 연동하여 시간표를 불러오는 기능이 동작하지 않는 경우예요.

**확인 사항:**
1. **인터넷 연결** 상태를 확인하세요.
2. **설정 → 학교 정보**에서 학교가 올바르게 등록되어 있는지 확인하세요.
3. **학년, 반 정보**가 정확히 입력되어 있는지 확인하세요.
4. NEIS API는 보통 **새벽 시간대(00:00~06:00)에 점검**이 이루어져요. 이 시간에는 일시적으로 시간표를 불러오지 못할 수 있어요. 잠시 후 다시 시도해 보세요.

**직접 입력 방법:**
- NEIS 연동이 계속 안 된다면 시간표를 **직접 입력**하는 방법도 있어요.
- 시간표 페이지에서 각 교시별 과목과 교사 정보를 직접 편집할 수 있어요.

---

### 급식 정보가 표시되지 않음

대시보드에 급식 정보가 나타나지 않는 경우예요.

**확인 사항:**
1. **인터넷 연결**이 필요해요. 오프라인 상태에서는 표시되지 않아요.
2. **설정에서 학교가 등록**되어 있는지 확인하세요.
3. **방학, 급식 미실시일(토요일, 공휴일 등)** 에는 급식 정보가 없어서 표시되지 않아요.
4. NEIS API 일시 장애로 급식 정보를 불러오지 못할 수 있어요. 잠시 후 다시 확인해 보세요.

## 화면/UI 문제

### 화면이 깨져 보이거나 글자가 겹침

레이아웃이 이상하게 표시되거나 텍스트가 겹쳐 보이는 경우예요.

**해결 방법:**
1. **앱을 종료하고 다시 실행**해 보세요. 대부분 이것으로 해결돼요.
2. **Windows 디스플레이 배율**을 확인하세요.
   - Windows 설정 → 디스플레이 → 배율
   - **100% 또는 125%** 로 설정하는 것을 권장해요.
   - 150% 이상의 고배율에서는 레이아웃이 다르게 보일 수 있어요.
3. **최신 버전으로 업데이트**되어 있는지 확인하세요. UI 버그는 업데이트를 통해 수정되는 경우가 많아요.

---

### 글자가 너무 작거나 큼

화면의 텍스트 크기가 적절하지 않은 경우예요.

**해결 방법:**
- Windows 설정 → 디스플레이 → 배율(Scale) 항목에서 배율을 조정하세요.
- **100~125%** 가 쌤핀에서 가장 보기 좋아요.
- 배율 변경 후 쌤핀을 재시작하면 변경 사항이 반영돼요.

---

### 위젯 모드가 열리지 않음

위젯 모드 버튼을 눌렀는데 아무 반응이 없거나 위젯 창이 나타나지 않는 경우예요.

**해결 방법:**
1. 작업 표시줄을 확인해 보세요. 위젯 창이 화면 밖으로 나갔을 수 있어요.
2. 작업 관리자에서 쌤핀 프로세스를 모두 종료하고 다시 실행하세요.
3. 그래도 안 된다면 최신 버전으로 업데이트해 보세요.

---

## 데이터 관련 문제

### 데이터가 사라졌어요

입력했던 시간표, 학생 정보, 메모 등의 데이터가 사라진 것처럼 보이는 경우예요.

**먼저 확인하세요:**
- 쌤핀의 모든 데이터는 `%APPDATA%\ssampin\data\` 폴더에 JSON 파일로 저장돼요.
- Windows 탐색기 주소창에 `%APPDATA%\ssampin\data` 를 입력해서 파일이 있는지 확인해 보세요.
- 파일이 있다면 앱을 완전히 재시작해 보세요.

**백업 복원 방법:**
- 이전에 백업을 만들어 두셨다면 **설정 → 데이터 관리 → 복원** 기능을 사용하세요.
- 백업 파일(.ssampin)을 선택하면 데이터가 복원돼요.

> 앱을 삭제해도 `%APPDATA%\ssampin\data\` 폴더는 그대로 남아요. 데이터 손실 걱정 없이 재설치하실 수 있어요!

---

### 새 컴퓨터로 데이터 옮기기

기존 컴퓨터의 데이터를 새 컴퓨터로 이전하는 방법이에요.

**방법 1: Google Drive 동기화 사용 (권장)**
1. 기존 컴퓨터에서 **설정 → Google Drive** 에서 동기화를 활성화하세요.
2. 새 컴퓨터에 쌤핀을 설치하세요.
3. 새 컴퓨터에서도 동일한 Google 계정으로 **설정 → Google Drive** 동기화를 활성화하세요.
4. 자동으로 데이터가 동기화돼요.

**방법 2: 백업 파일 사용**
1. 기존 컴퓨터에서 **설정 → 데이터 관리 → 백업** 을 실행하세요.
2. 백업 파일(.ssampin)을 USB나 이메일로 새 컴퓨터로 옮기세요.
3. 새 컴퓨터에 쌤핀을 설치하세요.
4. **설정 → 데이터 관리 → 복원** 에서 백업 파일을 선택하세요.

**방법 3: 데이터 폴더 직접 복사**
1. 기존 컴퓨터의 `%APPDATA%\ssampin\data\` 폴더 전체를 복사하세요.
2. 새 컴퓨터에 쌤핀을 설치하세요.
3. 새 컴퓨터의 동일 경로에 복사한 폴더를 붙여넣으세요.

---

### Google Drive 동기화 문제

Google Drive를 통한 데이터 동기화가 잘 안 되는 경우예요.

**확인 사항:**
1. **인터넷 연결** 상태를 확인하세요.
2. **구글 계정 로그인** 상태를 확인하세요. 토큰이 만료된 경우 재로그인이 필요해요.
3. **설정 → Google Drive** 에서 연결 상태를 확인하세요.

**재연결 방법:**
1. **설정 → Google Drive** 에서 **"연결 해제"** 를 클릭하세요.
2. 잠시 기다렸다가 **"Google Drive 연결"** 버튼을 클릭하세요.
3. Google 계정으로 다시 로그인하세요.
4. 동기화가 재개돼요.

---

### 데이터를 백업하고 싶어요

중요한 데이터를 잃지 않도록 백업하는 방법이에요.

**자동 백업 (Google Drive):**
- 설정 → Google Drive 동기화를 켜면 자동으로 클라우드에 백업돼요.

**수동 백업:**
- 설정 → 데이터 관리 → 백업 버튼을 클릭하면 .ssampin 파일로 내보낼 수 있어요.
- 이 파일을 안전한 위치에 저장해 두세요.

**직접 접근:**
- `%APPDATA%\ssampin\data\` 폴더를 직접 복사해서 보관할 수도 있어요.

## 성능 문제

### 앱이 느려요

쌤핀이 전반적으로 반응이 느린 경우예요.

**해결 방법:**
1. **앱을 종료하고 다시 실행**해 보세요. 장시간 실행 후에는 재시작이 도움이 돼요.
2. **최신 버전**인지 확인하세요. 성능 개선이 포함된 업데이트가 있을 수 있어요.
3. **오래된 데이터 정리**를 해보세요.
   - 완료된 할 일은 보관함으로 이동하거나 삭제하세요.
   - 오래된 메모는 정리하세요.
4. **다른 무거운 프로그램을 닫고** 쌤핀만 실행해 보세요.

### 앱이 갑자기 꺼짐

사용 중에 예고 없이 앱이 종료되는 경우예요.

**해결 방법:**
1. **최신 버전으로 업데이트**하세요. 알려진 크래시 버그가 수정되어 있을 수 있어요.
2. 어떤 동작을 했을 때 꺼지는지 패턴을 파악해 두세요.
3. **반복적으로 발생한다면** AI 도우미의 에스컬레이션 기능을 통해 개발자에게 버그 리포트를 보내주세요. 어떤 상황에서 발생했는지 최대한 자세히 적어주시면 빠른 수정에 도움이 돼요.

## 기타 자주 묻는 질문

### 쌤핀은 무료인가요?

네, 쌤핀은 완전 무료예요! 교사분들을 위해 개인 개발자가 만든 오픈소스 앱이에요.

### 인터넷이 없어도 사용할 수 있나요?

대부분의 기능은 오프라인에서도 완전히 동작해요. 데이터는 모두 내 컴퓨터에 저장돼요.
단, 다음 기능은 인터넷 연결이 필요해요:
- NEIS 시간표/급식 정보 불러오기
- 날씨 정보 표시
- Google Drive 동기화
- 자동 업데이트

### 학생 정보가 외부로 전송되나요?

아니요. 학생 이름, 학번 등 모든 데이터는 내 컴퓨터(`%APPDATA%\ssampin\data\`)에만 저장돼요. 외부 서버로 전송되지 않아요. (Google Drive 동기화를 직접 설정한 경우는 제외)

### 쌤핀을 여러 컴퓨터에서 동시에 사용할 수 있나요?

Google Drive 동기화 기능을 활성화하면 여러 컴퓨터에서 동일한 데이터를 사용할 수 있어요. 단, 동시에 여러 컴퓨터에서 수정하면 충돌이 발생할 수 있으니 순차적으로 사용하는 것을 권장해요.

### 건의사항이나 버그를 어떻게 전달하나요?

- 앱 내 AI 도우미의 에스컬레이션 기능을 사용하세요.
- 또는 GitHub Issues에 직접 등록해 주세요.
- 개발자가 최대한 빠르게 반영하도록 노력할게요!
