## Thread 5 (답글 4 / 8장 중 5)
**첨부 이미지**: `cards/05-shortcuts.png`

```
4. 단축키 — 어디서든 키보드 한 번 ⭐

v2.0.0에서 단축키 시스템을 본격 도입했어요. 설정 > 단축키에서 자유롭게 변경 가능합니다.

• Ctrl+K — 명령 팔레트(페이지 이동·기능 실행 검색)
• Ctrl+B — 사이드바 접기/펼치기
• Ctrl+Alt+T — 빠른 추가: 할 일
• Ctrl+Alt+E — 빠른 추가: 일정
• Ctrl+Alt+M — 빠른 추가: 메모
• Ctrl+Alt+N — 빠른 추가: 노트
• Ctrl+Alt+B — 빠른 추가: 즐겨찾기
• Ctrl+Enter — 빠른 추가 모달 저장 / 댓글 전송

페이지별 단축키도 정리됐어요. 좌석배치·시간표·칠판은 Ctrl+Z/Y, 칠판 G(배경 순환)·E·Shift+E(지우개 전환), 위젯 Ctrl+1~4(레이아웃), 즐겨찾기 / 또는 Ctrl+F(검색)·Ctrl+G(그룹)·Ctrl+E(편집).

손이 마우스로 자주 가던 동선이 키 한두 번으로 줄어듭니다.
```

## Thread 6 (답글 5 / 8장 중 6)
**첨부 이미지**: `cards/06-bookmarks.png`

```
5. 즐겨찾기 강화 — 단순 링크에서 사이트 라이브러리로

즐겨찾기를 풍부한 사이트 카드로 다시 설계했습니다.

• 링크 OG 미리보기 — 추가만 하면 사이트 제목·설명·썸네일이 카드로 (open-graph-scraper 기반, 한국 주요 사이트 호환성 보강)
• 검색 — 제목·설명·URL 전체에서 즉시
• 아카이브 — 자주 안 쓰는 사이트는 정리, 언제든 복관
• JSON 가져오기/내보내기 — 다른 PC·동료 교사와 공유
• '잊고 있던 사이트' 섹션 — 오랫동안 안 들어간 곳 다시 발견
• 위젯 모드 + 글로벌 빠른 추가 통합 — 클립보드에 URL 있으면 자동 채움 + OG 제목 자동 와인

수업 자료·교육청 사이트가 한곳에 정돈되고, 사이드바 안 거치고 위젯에서도 등록이 끝나요.
```
