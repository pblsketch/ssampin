# 쌤핀(SsamPin) v1.10.1 릴리즈 노트

**릴리즈일**: 2026-04-17
**제품**: 쌤핀 — 한국 중·고등학교 교사용 데스크톱 대시보드 앱

## 하이라이트

1. 출결 편집 UX 개선 — 출석 상태에서도 안내 문구 표시
2. 담임메모 출결 기록에 교시별 뱃지 추가
3. 좌석배치 학생 이름 크기 설정

## 상세 변경사항

- **[개선] 출결 편집 UX 개선**
  출석 상태일 때 사유·메모 영역이 사라져 입력 불가로 오인되던 문제를 해결. 이제 '결석·지각·조퇴·결과로 변경하면 사유와 메모를 입력할 수 있어요'라는 안내 문구가 표시됩니다.

- **[신규] 담임메모 교시별 출결 뱃지**
  출결 기록에 어느 교시에 이상 출결이 있었는지 뱃지로 표시. 같은 유형이면 '1교시, 2교시, 3교시'처럼 교시만, 유형이 다르면 교시별 유형·사유까지 나타납니다.

- **[신규] 좌석배치 학생 이름 크기 조정**
  설정 > 좌석에서 학생 이름 글자 크기를 4단계(작게/보통/크게/아주 크게)로 조절 가능. 학생 수가 적은 학급에서 이름을 크게 표시하고 싶을 때 유용합니다.

- **[수정] 버전 분포 분석 최신 버전 누락**
  분석 대시보드의 버전 분포에서 limit 제한으로 최신 버전이 표시되지 않던 문제를 수정했습니다.
