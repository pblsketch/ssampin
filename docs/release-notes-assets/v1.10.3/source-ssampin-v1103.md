## 상세 변경사항
- **[신규·베타] 협업 보드** — 쌤도구에 추가, Excalidraw 캔버스 기반 QR 실시간 협업, 최대 50명, 30초 자동 저장, 강제 종료 후 드로잉 복원.
- **[신규·베타] 복합 설문 스프레드시트 뷰** — 결과 화면 3탭(요약/테이블/개별), 정렬·필터·검색, 키보드 ←/→ 내비, 익명 토글.
- **[신규·베타] Excel 내보내기** — 요약·전체응답·워드빈도 3시트, 파일명 자동 sanitize.
- **[신규·베타] 피드백 프리셋 3종** — "발제 퀵 피드백" / "사례나눔 리플렉션" / "소풍 활동 소감".
- **[신규·베타] 라이브 피드백 월 (FeedbackWallView)** — 프로젝터용 포스트잇 월, 실시간 카드 fade-in, 하이라이트·고정·장문 확대 모달.
- **[개선] 설문 결과 화면 안정화** — 구 ResultsView 약 480줄 제거, 도메인 순수 함수로 재작성, 52개 단위 테스트 추가.
- **[개선] 대시보드 위젯 글씨 크기 통일** — 학급시간표·자주 쓰는 도구 주 콘텐츠를 12px로 상향, 다른 위젯과 리듬 맞춤.
- **[변경] Vitest 테스트 환경 도입** — 회귀 방지 기반 (개발자용).
