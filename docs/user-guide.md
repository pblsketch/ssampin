# 쌤핀(SsamPin) 사용 가이드

> 교사를 위한 똑똑한 데스크톱 대시보드, 쌤핀의 모든 기능을 안내합니다.

---

## 시작하기

### 다운로드 및 설치

#### Windows
1. **다운로드**: [ssampin.com](https://ssampin.com)에서 Windows 설치 파일(ssampin-Setup.exe)을 다운로드합니다.
2. **설치**: 다운로드한 파일을 실행하면 자동으로 설치됩니다.
3. **실행**: 바탕화면의 "쌤핀" 아이콘을 클릭하여 실행합니다.

#### macOS
1. **다운로드**: [ssampin.com](https://ssampin.com)에서 macOS용 DMG 파일을 다운로드합니다. Apple Silicon(M1~M4)과 Intel Mac 모두 지원해요.
2. **설치**: DMG 파일을 열고, 쌤핀 아이콘을 "응용 프로그램" 폴더로 드래그합니다.
3. **실행**: Launchpad 또는 응용 프로그램에서 쌤핀을 클릭합니다.
4. **첫 실행 시 경고**: "개발자를 확인할 수 없음" 경고가 뜨면, 시스템 설정 → 개인정보 보호 및 보안에서 "확인 없이 열기"를 클릭하세요. 또는 앱을 Control+클릭 → "열기"를 선택하세요.

### Windows 설치 시 경고 해결

쌤핀은 개인 개발 앱이라 Microsoft 인증서가 아직 없어요. 설치 시 보안 경고가 나타날 수 있습니다.

- **"Microsoft Windows의 PC 보호" 화면**: "추가 정보"를 클릭한 후 "실행" 버튼을 누르세요.
- **"스마트 앱 컨트롤이 차단했습니다" 화면 (Windows 11)**: 설치 파일을 우클릭 → 속성 → "차단 해제" 체크 → 확인을 누른 후 다시 실행하세요.

이 경고는 바이러스가 아니라 서명되지 않은 앱에 대한 Windows의 기본 보호 기능입니다.

### 온보딩 마법사

첫 실행 시 온보딩 마법사가 자동으로 시작됩니다.

1. **학교 설정**: 학교급(초/중/고)을 선택하고, NEIS에서 학교를 검색하여 등록합니다.
2. **학급 정보**: 학년, 반 정보를 입력합니다.
3. **교시 설정**: 학교급에 맞는 교시 프리셋이 자동으로 적용됩니다. 시작/종료 시간을 커스텀할 수 있습니다.
4. **시간표 불러오기**: NEIS 시간표를 자동으로 가져오거나, 직접 입력할 수 있습니다.

## 대시보드

대시보드는 쌤핀의 메인 화면으로, 교사에게 필요한 정보를 한눈에 보여줍니다.

### 위젯 구성

대시보드에는 다양한 위젯 카드가 표시됩니다:

- **현재 시간**: 실시간 시계
- **현재 교시**: 설정된 시간표 기준으로 현재 교시와 과목을 자동 표시
- **다음 수업**: 다음 교시 과목 미리보기
- **오늘의 일정**: 등록된 학급 행사와 일정 (표시 기간을 기본 14일에서 사용자가 원하는 기간으로 변경 가능)
- **할일 목록**: 오늘의 할일 요약
- **D-Day**: 중요 일정까지 남은 일수 카운트다운
- **급식 정보**: NEIS API로 자동 조회된 오늘의 급식 메뉴
- **날씨**: 현재 날씨 정보 (전국 83개 시/군 지원, 인터넷 연결 필요)
- **한줄 메시지**: 교실 전광판처럼 메시지를 표시
  - 오늘의 한마디 배너의 요소별 색상(배경, 텍스트, 강조)을 자유롭게 변경할 수 있습니다.

### 자주 쓰는 쌤도구 위젯

- 대시보드와 위젯 모드에서 자주 사용하는 쌤도구(타이머, 랜덤 뽑기, 점수판 등)를 바로 실행할 수 있는 위젯입니다.
- 편집 버튼을 클릭하면 표시할 도구를 직접 선택할 수 있습니다 (최대 8개).
- 도구를 클릭하면 해당 도구 페이지로 바로 이동합니다.

### 미니 캘린더 위젯

대시보드에서 이번 달 캘린더를 한눈에 볼 수 있는 미니 위젯입니다.

- 일정이 있는 날짜에 점 표시로 빠르게 확인할 수 있습니다.
- 날짜를 클릭하면 해당 날짜의 일정이 팝업으로 표시되어 바로 확인할 수 있습니다.

### 이미지 스티커 위젯

대시보드에 이미지 스티커를 추가하여 나만의 대시보드를 꾸밀 수 있습니다 (다꾸 컨셉).

- 대시보드 편집 모드에서 **이미지 스티커 추가** 버튼으로 이미지를 업로드합니다.
- .jpg, .png, .svg, .webp, .gif 형식을 지원합니다.
- 스티커 크기와 위치를 자유롭게 조절할 수 있습니다.
- 여러 개의 스티커를 추가하여 대시보드를 개성 있게 꾸며보세요.

### 위젯 커스터마이즈

- 위젯 카드를 우클릭하면 표시/숨김을 설정할 수 있습니다.
- 위젯의 배치 순서를 변경할 수 있습니다.
- **그리드 행 높이 조절**: 위젯 편집 모드에서 그리드 행 높이를 슬라이더로 자유롭게 조절할 수 있습니다. 위젯 카드의 세로 크기를 원하는 대로 설정하세요.

### 글씨 크기 조절

대시보드와 위젯 모드에서 글씨 크기를 조절할 수 있습니다.

1. 대시보드 우측 상단의 **글씨 크기 조절 슬라이더**를 사용합니다.
2. 위젯 편집 패널에서도 글씨 크기를 별도로 조절할 수 있습니다.
3. 프로젝터나 대형 화면에서는 큰 글씨로, 개인 모니터에서는 작은 글씨로 설정하세요.

> **활용 팁**: 수업 중 프로젝터에 위젯을 띄울 때 글씨 크기를 키우면 교실 뒤에서도 잘 보여요!

## 시간표

### NEIS 자동 연동

1. 설정에서 학교를 검색하여 등록하면 NEIS API를 통해 시간표를 자동으로 가져옵니다.
2. 학년, 반 정보를 입력하면 해당 학급의 시간표가 자동으로 표시됩니다.
3. 시간표는 주기적으로 업데이트되며, 수동으로 새로고침할 수도 있습니다.
4. 시간표 페이지에서 편집 모드 없이 바로 **NEIS 불러오기** 버튼으로 시간표를 가져올 수 있습니다.

### 직접 입력

NEIS 연동을 사용하지 않거나, 시간표를 수정하고 싶을 때:

1. 시간표 페이지에서 빈 칸을 클릭합니다.
2. 과목명을 입력하거나 목록에서 선택합니다.
3. 저장하면 즉시 반영됩니다.

### 과목별 컬러코드

시간표에 과목별로 자동으로 색상이 지정됩니다:

| 과목 | 색상 |
|------|------|
| 국어 | 노란색 (Yellow) |
| 영어 | 초록색 (Green) |
| 수학 | 파란색 (Blue) |
| 과학 | 보라색 (Purple) |
| 사회 | 주황색 (Orange) |
| 체육 | 빨간색 (Red) |
| 음악 | 분홍색 (Pink) |
| 미술 | 남색 (Indigo) |
| 창체 | 청록색 (Teal) |

### 과목별 색상 커스터마이징

- 기본 색상 외에 각 과목의 색상을 직접 변경할 수 있습니다.
- 인라인 팔레트에서 16가지 프리셋 색상 중 선택하거나, 기본값으로 복원할 수 있습니다.

### 시간표 색상 모드

- **과목별 색상**: 국어, 영어, 수학 등 과목 이름에 따라 색상이 자동 지정됩니다.
- **학반별 색상**: 수업반(반 번호)에 따라 색상이 구분됩니다.
- 설정에서 원하는 모드를 선택할 수 있습니다.

### NEIS 시간표 자동 연동 — 상세 가이드

NEIS에서 시간표를 자동으로 가져오는 전체 과정을 안내합니다.

1. 사이드바에서 **설정**을 클릭합니다.
2. **"학교 정보" 탭**을 선택합니다.
3. 학교 검색란에 **학교 이름**을 입력하고 **"검색"** 버튼을 클릭합니다.
4. 검색 결과에서 **내 학교를 선택**합니다. 같은 이름의 학교가 여러 개라면 주소를 확인하세요.
5. **학년**과 **반** 정보를 입력합니다.
6. **"저장"** 버튼을 클릭합니다.
7. 사이드바에서 **"시간표"** 를 클릭하면 NEIS에서 가져온 시간표가 자동으로 표시됩니다.

**시간표가 안 나올 때 확인 사항:**
- 인터넷 연결이 되어 있는지 확인하세요.
- NEIS API 점검 시간(보통 새벽 00:00~06:00)에는 일시적으로 데이터를 가져오지 못할 수 있어요.
- 학년, 반 정보가 정확한지 다시 확인하세요.
- 방학 중이거나 시간표 미등록 학교인 경우 NEIS에 데이터가 없을 수 있어요.

### 교시 시간 인라인 편집

시간표 편집 모드에서 교시별 시작·종료 시간을 직접 수정할 수 있습니다.

- 시간표 왼쪽의 교시 시간 영역을 클릭하면 인라인 편집 모드가 활성화됩니다.
- 시작 시간과 종료 시간을 바로 변경할 수 있습니다.
- 설정 페이지에 가지 않고도 시간표 화면에서 바로 교시 시간을 조정할 수 있어 편리합니다.

### 시간표 직접 입력 — 상세 가이드

NEIS 연동 없이 시간표를 직접 만드는 방법입니다.

1. 사이드바에서 **"시간표"** 를 클릭합니다.
2. 시간표 화면에서 **원하는 요일의 교시 칸을 클릭**합니다.
3. **과목명**을 입력합니다. 이전에 입력한 과목이 자동완성으로 나타날 수 있습니다.
4. 필요하면 **교사명**도 함께 입력합니다.
5. **Enter** 또는 확인 버튼을 눌러 저장합니다.
6. 다른 교시도 동일하게 채워 넣습니다.

**수정 방법**: 이미 입력된 칸을 다시 클릭하면 내용을 수정할 수 있습니다.
**삭제 방법**: 과목명을 비우고 저장하면 해당 교시가 빈 칸으로 돌아갑니다.

> **팁**: NEIS에서 가져온 시간표도 직접 수정할 수 있어요. 보강이나 시간표 변경이 있을 때 활용하세요.

### 교사 시간표 사용 방법

담임 시간표 외에 교사 본인의 전체 수업 시간표를 관리할 수 있습니다.

1. 시간표 페이지 상단에서 **"교사 시간표" 탭**을 선택합니다.
2. 각 교시에 **수업반(예: 2-3)**과 **과목**을 입력합니다.
3. 교사 시간표는 대시보드의 "현재 교시" 위젯에도 반영됩니다.

> **활용 팁**: 담임 시간표와 교사 시간표를 모두 설정하면, 대시보드에서 현재 어느 반에서 무슨 과목을 가르치는지 자동으로 표시돼요.

### 시간표 색상 모드 전환 방법

시간표 색상을 과목별 또는 학반별로 전환하는 방법입니다.

1. 시간표 페이지 상단의 **색상 모드 전환 버튼**을 클릭합니다.
2. **"과목별 색상"** 을 선택하면 국어=노란색, 영어=초록색 등 과목에 따라 색이 지정됩니다.
3. **"학반별 색상"** 을 선택하면 같은 반의 수업이 같은 색으로 표시됩니다.
4. 선택 즉시 시간표에 반영됩니다.

### 과목 색상 직접 변경하기

기본 색상이 마음에 들지 않으면 직접 바꿀 수 있습니다.

1. 시간표에서 **과목이 입력된 칸을 클릭**합니다.
2. 편집 화면 하단에 **인라인 팔레트**가 나타납니다.
3. **16가지 프리셋 색상** 중 원하는 색을 클릭합니다.
4. 변경된 색상은 해당 과목의 모든 칸에 동시에 적용됩니다.
5. 기본 색상으로 되돌리려면 팔레트 옆의 **"기본값" 버튼**을 클릭합니다.

### 시간표 새로고침 방법

NEIS 시간표를 수동으로 다시 가져오는 방법입니다.

1. 시간표 페이지에서 **새로고침 아이콘(🔄)** 을 클릭합니다.
2. 인터넷이 연결되어 있으면 NEIS에서 최신 시간표를 다시 가져옵니다.
3. 시간표 변경이 감지되면 자동으로 업데이트됩니다.

> **참고**: 직접 수정한 내용은 NEIS 새로고침 시 덮어쓰기될 수 있으니, 보강 등 임시 변경은 NEIS 새로고침 전에 다시 확인하세요.

### 시간표 임시 변경 (날짜별 오버라이드)

특정 날짜에 시간표를 임시로 변경할 수 있습니다. 원래 시간표는 유지됩니다.

1. 시간표 페이지에서 **임시 변경할 날짜**를 선택합니다.
2. 해당 날짜의 교시를 클릭하여 **다른 과목으로 변경**합니다.
3. 임시 변경된 교시는 별도 표시되며, 원래 시간표로 되돌릴 수 있습니다.
4. 보강, 시험감독, 행사 등으로 일시적으로 시간표가 바뀔 때 유용합니다.

> **참고**: 임시 변경은 해당 날짜에만 적용되며, 기본 시간표에는 영향을 주지 않습니다.

## 자리 배치

### 학생 명단 등록

1. 좌석배치 페이지에서 학생 명단을 직접 입력하거나, Excel 파일에서 불러올 수 있습니다.
   - Excel 업로드 시 헤더가 자동으로 감지됩니다.
   - .xls 형식은 .xlsx로 변환 후 업로드하라는 안내가 표시됩니다.
   - 업로드 전 미리보기로 데이터를 확인할 수 있습니다.
2. 학생 수와 교실 배치(행×열)를 설정합니다.

### 랜덤 배치

- "랜덤 배치" 버튼을 클릭하면 학생들이 무작위로 좌석에 배치됩니다.
- 특정 학생의 자리를 고정하거나, 짝 관계(같이 앉기/떨어져 앉기)를 설정할 수 있습니다.

### 조건부 랜덤 배치

랜덤 배치 시 다양한 조건을 지정할 수 있습니다:

- **구역 지정**: 특정 학생을 앞줄, 뒷줄 등 특정 구역에 배치
- **분리**: 함께 앉히지 않아야 할 학생 쌍 지정
- **인접**: 반드시 가까이 앉혀야 할 학생 쌍 지정
- **고정 좌석**: 특정 학생을 지정한 자리에 고정

### 드래그 앤 드롭

- 학생 이름을 드래그하여 원하는 자리로 이동할 수 있습니다.
- 두 학생의 자리를 서로 교환할 수 있습니다.

### 짝꿍 모드

- 2인 1조 짝꿍 배치를 자동으로 생성할 수 있습니다.
- **3인짝 모드**: 홀수 인원이거나 3인 한 조로 구성이 필요할 때 3인 짝꿍 배치를 지원합니다.
- **짝꿍 그룹 유지 셔플**: 기존 짝꿍 조합을 유지하면서 조별 위치만 셔플할 수 있습니다.
- 짝꿍 관계가 좌석배치표 내보내기에도 반영됩니다.

### 모둠(그룹) 레이아웃

좌석 배치에서 모둠별로 학생을 묶어 배치할 수 있습니다.

- **모둠 모드 활성화**: 좌석배치 페이지에서 모둠 레이아웃 모드를 선택합니다.
- **모둠 구성**: 학생들을 원하는 모둠으로 그룹핑합니다.
- **위젯 반영**: 좌석배치 위젯에서도 모둠 모드가 정확하게 표시됩니다.

> **활용 팁**: 조별 활동이 많은 수업에서는 모둠 모드로 좌석을 배치하면 편리해요.

### 교사 시점 보기

- **교사 시점 모드**를 활성화하면 교사가 학생들을 바라보는 방향에서 좌석배치를 확인할 수 있습니다.
- 칠판을 기준으로 좌우를 반전하여 표시합니다.

### 내보내기

좌석 배치표를 다양한 형식으로 내보낼 수 있습니다:

- **HWPX**: 한글(한컴오피스) 문서로 내보내기
- **XLSX**: Excel 스프레드시트로 내보내기
- **PDF**: PDF 문서로 내보내기
- **좌우 반전 옵션**: Excel/한글 내보내기 시 교사 시점으로 좌우를 반전하여 출력 가능

쌤도구는 수업에 바로 쓸 수 있는 15가지 교실 도구 모음입니다.

### 도구 목록

| 도구 | 설명 |
|------|------|
| ⏱️ **타이머** | 시간 제한 활동에 사용. 시간을 설정하고 카운트다운을 시작합니다. 시간 조정(+/-) 버튼으로 실시간 조절 가능. |
| 🎲 **랜덤 뽑기** | 학생 명단에서 무작위로 발표자를 선정합니다. 공정한 기회 보장. |
| 🚦 **신호등** | 빨강/노랑/초록으로 활동 시작과 멈춤을 시각적으로 안내합니다. |
| 📊 **점수판** | 팀별 점수를 관리합니다. 팀 이름과 점수를 자유롭게 설정. |
| 🎯 **룰렛** | 항목을 추가하고 돌려서 결정합니다. |
| 🎲 **주사위** | 주사위를 굴려 결과를 확인합니다. |
| 🪙 **동전** | 동전 던지기로 앞/뒤를 결정합니다. |
| 🔗 **QR코드** | URL을 입력하면 QR코드를 생성하여 학생들과 빠르게 링크를 공유합니다. |
| 🤫 **활동 기호** | 수업 모드(조용히, 토론, 짝활동 등)를 한눈에 표시합니다. |
| 📊 **투표** | 선택지를 만들어 학생들의 의견을 모읍니다. 실시간 결과 확인. |
| 📝 **주관식 설문** | 학생들이 자유롭게 의견을 입력할 수 있는 설문을 만듭니다. |
| ☁️ **워드클라우드** | 학생들이 입력한 단어를 모아 워드클라우드로 시각화합니다. |
| 🪑 **자리 뽑기** | 학생이 직접 자리를 뽑을 수 있는 인터랙티브 도구입니다. |
| 🌳 **숲소리** | 교육 웹진 숲소리(supsori.com)로 바로 이동합니다. |
| 🎯 **PBL스케치** | 수업 및 평가 설계 도구(pblsketch.xyz)로 바로 이동합니다. |

### 타이머 예고 알림

- 타이머 종료 전에 미리 알림을 받을 수 있습니다.
- 예고 알림 시간(예: 30초 전, 1분 전)을 설정할 수 있습니다.
- 알림 시 화면에 배너가 표시되어 학생들도 시간이 얼마 남지 않았음을 알 수 있습니다.

### 프리셋 저장

- 자주 사용하는 도구 설정(타이머 시간, 투표 항목 등)을 프리셋으로 저장할 수 있습니다.
- 저장된 프리셋은 다음에 바로 불러올 수 있어 수업 준비 시간을 절약합니다.

### 학생 참여형 도구 (투표/설문/워드클라우드)

투표, 주관식 설문, 워드클라우드 도구는 학생들이 스마트폰으로 참여할 수 있습니다:

1. 도구를 시작하면 참여 QR코드가 생성됩니다.
2. 학생들이 QR코드를 스캔하여 참여합니다.
3. 교사의 화면에 실시간으로 결과가 표시됩니다.

## 수업 관리

수업 관리는 수업반(클래스)별로 출석, 진도, 과제수합, 설문을 관리하는 기능입니다.

### 수업반 관리

- 담당하는 수업반(예: 2학년 3반 수학)을 생성하고 관리할 수 있습니다.
- 수업반에 학년/반 소속 명렬표를 연동하거나 직접 학생 명단을 입력할 수 있습니다.
- 수업반별로 별도의 좌석배치, 진도, 출결 기록이 관리됩니다.
- 수업반을 추가하면 교사 시간표에서 해당 수업 정보가 자동으로 연동됩니다.
- 학생의 상태(재학/전출/결번)를 관리할 수 있으며, 결번 처리된 학생의 이름이 유지됩니다.

### 출석 관리

- 학생별 출석 상태(출석/결석/지각/조퇴)를 체크할 수 있습니다.
- 날짜별 출석 기록이 자동으로 저장됩니다.
- 출결 기록은 유형별 태그 색상으로 구분되어 한눈에 파악할 수 있습니다.
- 정렬 기능으로 특정 출결 유형만 모아볼 수 있습니다.
- 출결 기록을 **엑셀 파일로 내보내기**할 수 있습니다.
- 시간표가 연동된 경우, 현재 교시가 교시 버튼에서 **하이라이트** 표시됩니다.
- 달력에 **수업일이 표시**되어 수업이 있는 날짜를 한눈에 파악할 수 있습니다.

### 진도 관리

- 수업별 진도를 기록하고 관리할 수 있습니다.
- 진도 편집 폼에 필드 라벨(단원, 내용, 메모 등)이 표시되어 입력이 편리합니다.
- 캘린더에서 과목별 색상이 적용되어 진도 일정을 한눈에 구분할 수 있습니다.
- 다른 반의 진도를 불러와서 날짜만 수정하여 적용할 수 있습니다.

### 과제수합

- 학생들에게 파일 또는 텍스트로 과제를 제출받을 수 있습니다.
- 과제 생성 시 제출 방식(파일/텍스트/둘 다)을 선택할 수 있습니다.
- QR코드 또는 URL을 공유하여 학생들이 스마트폰으로 바로 제출할 수 있습니다.
- **Google Drive 연동**: 파일 제출물을 Google Drive에 자동으로 저장합니다.
- 교사는 제출 현황을 실시간으로 확인하고, 텍스트 제출물은 팝업으로 확인할 수 있습니다.
- 제출 현황을 엑셀로 내보낼 수 있습니다.

### 설문/체크리스트

- 학생 응답 수집을 위한 설문 또는 교사가 직접 체크하는 체크리스트를 만들 수 있습니다.
- **교사 모드**: 교사가 화면에서 직접 체크하는 방식
- **학생 응답 모드**: QR코드 또는 URL을 공유하여 학생이 스마트폰으로 직접 응답
- **PIN 코드 인증**: 학생 응답 시 본인 확인 PIN을 설정하여 사칭을 방지할 수 있습니다.
- 학생별 메모를 추가할 수 있습니다.

## 담임 업무

담임 업무는 담임 교사의 학급 관리를 위한 통합 허브입니다. 5개의 탭으로 구성되어 있습니다.

### 학생 기록

학생별 상담, 행동, 학업 등 다양한 카테고리의 기록을 관리합니다.

- **카테고리별 기록**: 상담, 행동, 학업, 출결 등 칩(태그)으로 분류
- **인라인 편집**: 칩을 클릭하여 바로 편집할 수 있습니다
- **기간/카테고리/학생 필터**: 원하는 조건으로 기록을 검색하고 조회할 수 있습니다
- **내보내기**: 기록된 내용을 다양한 형식으로 내보낼 수 있습니다
  - **Excel**: 학생별 기록을 스프레드시트로 내보내기
  - **HWPX (한글)**: 한컴오피스 문서로 내보내기
  - **생활기록부용 Excel**: 생활기록부 작성에 최적화된 형식으로 내보내기
- 조회 탭에서 검색 결과를 바로 Excel로 내보낼 수 있습니다

### 설문/체크리스트

수업 관리의 설문/체크리스트 기능과 동일하게 담임 학급을 대상으로 사용합니다. (학생 응답 모드, PIN 코드 인증, 학생별 메모 지원)

### 상담 예약

학생 및 학부모의 상담을 온라인으로 예약받을 수 있습니다.

- **위자드 형태 UI**: 단계별로 안내되는 직관적인 상담 일정 생성 화면
- **시간표 연동**: 수업 시간은 자동으로 차단되어 상담 불가 슬롯으로 표시됩니다
- **예약 슬롯 자동 차단**: 이미 예약된 시간은 자동으로 차단됩니다
- **예약자 정보 수정**: 예약 후에도 예약자 정보를 수정할 수 있습니다
- **1인당 상담 시간 설정**: 상담 1건당 소요 시간을 설정할 수 있습니다
- **개인정보 보호 토글**: 예약자 명단을 보호 모드로 전환할 수 있습니다
- **QR/URL 공유**: 생성된 상담 예약 링크를 QR코드 또는 URL로 학생/학부모와 공유합니다

### 과제 수합

학급 학생들에게 과제를 제출받습니다. Google Drive 연동 및 QR/URL 공유를 지원합니다. (수업 관리의 과제수합과 동일한 기능)

### 자리배치

담임 교실의 좌석배치를 관리합니다. 자리 배치 기능의 모든 기능(랜덤 배치, 조건부 배치, 짝꿍 모드 등)을 그대로 사용할 수 있습니다.

### 학생 생일 → 일정 동기화

학생 명단에 생년월일을 등록하면 생일을 일정에 자동으로 동기화할 수 있습니다.

1. 좌석배치 또는 수업 관리에서 학생 명단의 **생년월일** 필드를 입력합니다.
2. 일정 관리에서 **생일 동기화** 토글을 켜면 학생 생일이 캘린더에 자동 등록됩니다.
3. 생일 일정은 매년 반복되며, 별도 카테고리로 구분됩니다.

## 위젯 모드

위젯 모드는 대시보드를 작은 창으로 표시하여 다른 작업을 하면서도 정보를 확인할 수 있는 모드입니다.

### 사용 방법

1. 사이드바 하단의 "위젯" 버튼을 클릭합니다.
2. 별도의 작은 창이 열리며, 대시보드의 핵심 정보가 표시됩니다.

### 주요 기능

- **일반 모드**: 기본 모드로, 다른 창에 가려질 수 있습니다.
- **항상 위 모드**: 다른 프로그램 위에 항상 표시됩니다. 수업 중 항상 확인이 필요할 때 유용합니다.
- **드래그 이동**: 창을 드래그하여 원하는 위치로 이동할 수 있습니다.
- **레이아웃 변경**: 전체(Full), 가로 분할(Split-H), 세로 분할(Split-V), 4분할(Quad) 모드를 선택할 수 있습니다. 레이아웃 드롭다운에 키보드 단축키(Ctrl+1~4)가 함께 표시됩니다.
- **위젯 선택**: 표시할 위젯을 선택하여 원하는 정보만 볼 수 있습니다.
- **편집 모드**: 위젯 순서를 변경하거나 표시/숨김을 설정할 수 있습니다.
- **D-Day 위젯**: 중요 일정까지 남은 일수를 위젯으로 표시합니다.
- **메모 포커스 위젯**: 특정 메모를 포커스하여 표시하며, 이전/다음 네비게이션으로 여러 메모를 순서대로 확인할 수 있습니다.
- **자주 쓰는 쌤도구 위젯**: 자주 사용하는 교실 도구를 위젯에서 바로 실행할 수 있습니다.
- **즐겨찾기 표시 항목 편집**: 즐겨찾기 위젯에서 표시할 항목을 직접 선택하고 편집할 수 있습니다.
- **날씨 정보 표시**: 위젯 모드 헤더에 현재 온도, 습도, 미세먼지, 날씨 상태가 표시됩니다. 인터넷 연결 시 자동으로 업데이트됩니다.

### 위젯 모드 열기 — 단계별 가이드

위젯 모드를 처음 사용하는 분을 위한 상세 안내입니다.

1. 쌤핀 메인 앱의 **사이드바 하단**을 확인하세요. "위젯" 버튼이 있습니다.
2. **"위젯" 버튼을 클릭**하면 별도의 작은 창이 화면에 나타납니다.
3. 메인 앱 창과 위젯 창은 **독립적으로 동작**합니다. 메인 앱을 닫아도 위젯 창은 유지됩니다.
4. 위젯 창 상단의 **핀 아이콘(항상 위)** 을 클릭하면 다른 프로그램 위에 항상 표시됩니다.
5. 위젯 창을 닫으려면 **창 상단의 X 버튼**을 클릭하거나, **우클릭 메뉴에서 "닫기"** 를 선택하세요.

> **팁**: 수업 중에 시간표와 현재 교시를 항상 보고 싶다면 "항상 위" 모드를 켜두는 것을 추천해요!

### 위젯 레이아웃 변경 방법

위젯 창의 레이아웃을 원하는 형태로 변경할 수 있습니다.

1. 위젯 창 상단의 **레이아웃 아이콘**을 클릭하세요.
2. 4가지 레이아웃 중 선택합니다:
   - **전체(Full)**: 하나의 위젯을 크게 표시
   - **가로 분할(Split-H)**: 좌우로 2개의 위젯 표시
   - **세로 분할(Split-V)**: 상하로 2개의 위젯 표시
   - **4분할(Quad)**: 4개의 위젯을 동시에 표시
3. 각 레이아웃 옆에 **키보드 단축키(Ctrl+1~4)**가 표시됩니다.
4. 창 크기가 너무 작으면 자동으로 "전체" 모드로 전환됩니다.

> **팁**: 키보드 단축키로 빠르게 레이아웃을 변경할 수 있어요! Ctrl+1(전체), Ctrl+2(좌우), Ctrl+3(상하), Ctrl+4(4분할), Ctrl+0(순환)

> **주의**: 창 너비가 640px 미만이면 자동으로 전체 모드가 적용돼요.

### 위젯에 표시할 항목 선택하기

원하는 정보만 골라서 위젯에 표시할 수 있습니다.

1. 위젯 창에서 **편집 모드 아이콘(연필 모양)** 을 클릭하세요.
2. 편집 패널이 열리면 표시할 위젯을 **토글(켜기/끄기)** 로 선택합니다.
3. 위젯 순서를 **드래그하여 변경**할 수 있습니다.
4. 선택이 끝나면 편집 모드를 다시 클릭하여 닫습니다.
5. 변경 사항은 즉시 반영됩니다.

선택 가능한 위젯: 현재 시간, 현재 교시, 다음 수업, 오늘의 일정, 할일, D-Day, 급식, 날씨, 한줄 메시지, 메모 포커스, 자주 쓰는 쌤도구, 즐겨찾기 등

### 위젯 스타일 커스터마이징

위젯의 외관을 자유롭게 꾸밀 수 있습니다. 위젯 편집 패널 또는 설정 → 디스플레이 탭에서 설정합니다.

- **테마 프리셋**: 노션 라이트/다크, 크래프트 노트 라이트/다크 등 다양한 프리셋을 선택할 수 있습니다.
- **투명도 조절**: 위젯 배경의 투명도를 자유롭게 조절합니다.
- **테두리 설정**: 테두리의 표시 여부, 두께, 모서리 곡률을 세부적으로 설정합니다.
- **배경 패턴**: 카드 배경에 패턴을 적용할 수 있습니다.
- **강조 색상**: 위젯의 강조(액센트) 색상을 스와치에서 선택합니다.

> **활용 팁**: 프로젝터에 띄울 때는 라이트 테마 프리셋으로, 개인 모니터에서는 다크 테마로 사용하면 좋아요!

> **참고**: 라이트 테마에서 시간표 과목 텍스트의 가독성이 개선되었습니다 (WCAG AA 기준 충족). 프로젝터 사용 시 더 선명하게 보입니다.

### 위젯 항상 위(Always on Top) 모드 설정

다른 프로그램 위에 위젯을 항상 표시하는 방법입니다.

1. 위젯 창 상단 바에서 **핀(📌) 아이콘**을 클릭하세요.
2. 핀이 활성화되면 아이콘 색상이 변합니다.
3. 이제 다른 프로그램 창을 클릭해도 위젯이 가려지지 않습니다.
4. 다시 핀 아이콘을 클릭하면 항상 위 모드가 해제됩니다.

> **활용 팁**: PPT 수업 중에 위젯을 화면 구석에 두면 시간표와 타이머를 동시에 확인할 수 있어요.

### 위젯 창 이동 및 크기 조절

1. 위젯 창 **상단 바를 드래그**하면 원하는 위치로 이동할 수 있습니다.
2. 위젯 창의 **모서리나 가장자리를 드래그**하면 크기를 조절할 수 있습니다.
3. 위치와 크기는 자동 저장되어 다음에 열 때도 동일한 위치에 나타납니다.

### 위젯 모드 닫기

위젯 모드를 종료하는 방법입니다.

1. 위젯 창 **우측 상단의 X 버튼**을 클릭합니다.
2. 또는 위젯 창에서 **우클릭 → "닫기"** 를 선택합니다.
3. 위젯 창을 닫아도 메인 앱에는 영향이 없습니다.
4. 다시 열려면 메인 앱 사이드바에서 "위젯" 버튼을 클릭하세요.

### 메인 앱 ↔ 위젯 실시간 동기화

메인 앱에서 메모, 할일, 일정 등을 수정하면 위젯에 즉시 반영됩니다. 앱을 재시작하지 않아도 항상 최신 데이터를 확인할 수 있습니다.

- 위젯 창이 열린 상태에서 메인 앱의 데이터를 변경하면 자동 동기화됩니다.
- **시간표도 실시간 동기화**: 메인 앱에서 시간표를 수정하면 위젯 모드에도 즉시 반영됩니다.
- 일정 위젯은 위젯 크기에 맞게 표시 개수가 자동으로 조정됩니다.

### 위젯 모드 자주 묻는 질문

**Q: 위젯 창이 화면에 안 보여요.**
A: 작업 표시줄에서 쌤핀 아이콘을 확인하세요. 위젯 창이 화면 밖으로 나간 경우, 메인 앱을 재시작하면 위치가 초기화됩니다.

**Q: 위젯에 시간표가 안 나와요.**
A: 메인 앱에서 시간표가 정상적으로 설정되어 있는지 확인하세요. 시간표 데이터가 없으면 위젯에도 표시되지 않습니다.

**Q: 위젯 창 크기를 아무리 줄여도 한계가 있어요.**
A: 최소 크기 제한이 있어요. 너무 작으면 정보가 읽기 어려워지므로 일정 크기 이하로는 줄어들지 않습니다.

**Q: 위젯에서 수정한 내용이 메인 앱에도 반영되나요?**
A: 네, 메인 앱과 위젯은 실시간으로 동기화됩니다. 어느 쪽에서든 수정하면 양쪽 모두 즉시 반영됩니다.

## 메모

### 포스트잇 메모

- 다양한 색상의 포스트잇 스타일 메모를 작성할 수 있습니다.
- 드래그하여 위치를 변경할 수 있습니다.
- **크기 조절(리사이즈)**: 메모 모서리를 드래그하여 크기를 자유롭게 조절할 수 있습니다.
- **기본 크기 복원**: 리사이즈 후 버튼 한 번으로 기본 크기로 되돌릴 수 있습니다.
- 서식 도구(굵게, 기울임, 밑줄 등)로 텍스트를 꾸밀 수 있습니다.
- 메모 색상을 변경할 수 있습니다.

### 위젯 모드에서 메모 네비게이션

- 위젯 모드에서 메모 포커스 위젯을 사용할 때, 이전/다음 버튼으로 여러 메모를 순서대로 확인할 수 있습니다.

### 학생 기록 (담임 업무)

학생별 상담 기록, 특이사항, 관찰 내용 등을 "담임 업무" 메뉴에서 관리합니다.

#### 입력 모드 — 3컬럼 레이아웃

입력 화면이 3컬럼 레이아웃으로 구성되어 효율적으로 기록할 수 있습니다.

- **왼쪽 (학생 목록)**: 번호순 또는 자리배치 뷰로 학생을 선택합니다.
- **가운데 (입력 폼)**: 카테고리, 내용, 상담방법 등을 입력합니다.
- **오른쪽 (참조 패널)**: 두 가지 탭으로 이전 기록을 참조할 수 있습니다.
  - **"오늘" 탭**: 선택한 학생의 오늘 기록을 바로 확인
  - **"이력" 탭**: 과거 기록을 펼쳐보고 인라인으로 수정 가능
- 컬럼 사이의 구분선을 드래그하여 너비를 자유롭게 조절할 수 있습니다.

#### 진행현황 대시보드

학급 전체의 기록 현황을 한눈에 파악할 수 있는 대시보드입니다.

- **기록 통계**: 기록 없는 학생 수, 학생당 평균 기록 건수
- **나이스 미반영 현황**: 출결 기록 중 나이스에 아직 반영하지 않은 건을 학생별로 드릴다운
- **상담방법 분석**: 상담 유형별(대면, 전화, 온라인 등) 건수 확인
- **후속조치 추적**: 기한이 지난 후속조치와 다가오는 후속조치를 시각적으로 관리
- **주의 학생 알림**: 기록 없는 학생, 출결만 있는 학생, 후속조치 미이행 학생을 카드로 표시

#### 검색 모드

강력한 필터와 분석 도구로 기록을 검색합니다.

- **액션 대시보드**: 나이스 미반영, 후속조치 현황을 클릭하면 바로 필터 적용
- **필터 요약 스트립**: 현재 검색 결과의 카테고리·학생 분포와 날짜 범위를 시각적으로 표시
- **학생 점프 리스트**: 사이드바에서 학생 이름을 클릭하면 해당 학생 기록으로 바로 이동. 미반영·미이행 경고 배지 표시
- **커스텀 날짜 범위**: "직접 지정" 옵션으로 원하는 기간의 기록을 검색

#### 기타 기능

- 카테고리 칩을 클릭하여 기록 유형을 빠르게 선택하고 인라인으로 편집할 수 있습니다.
- 날짜별로 기록이 저장되어 이력 관리가 가능합니다.
- PIN 잠금으로 민감한 정보를 보호할 수 있습니다.

## 할일 관리

### 할일 작성

- "+" 버튼으로 새 할일을 추가합니다.
- 중요도를 설정할 수 있습니다.
- 마감기한은 선택 사항입니다 — 마감일이 없는 할일도 자유롭게 등록할 수 있습니다.

### 할일 관리

- 체크박스를 클릭하여 완료 처리합니다.
- 마감일이 지난 할일은 자동으로 "지연됨" 표시가 됩니다.
- 드래그하여 순서를 변경할 수 있습니다.
- 할일 항목에 마우스를 올리면 **편집 아이콘(✏️)**이 표시됩니다. 클릭하면 상세 편집 모달이 열립니다.
- 할일 제목을 클릭하면 인라인으로 바로 수정할 수 있습니다.
- 완료된 할일을 접어서 깔끔하게 관리할 수 있습니다.

> **팁**: 처음 사용 시 편집 방법을 안내하는 배너가 표시됩니다. 확인 후 자동으로 사라집니다.

### 미루기

- 할일을 "내일", "다음 주", "직접 선택" 등으로 미룰 수 있습니다.
- 미루기 드롭다운 메뉴에서 빠르게 날짜를 변경합니다.

### 서브태스크

- 큰 할일 아래에 세부 하위 항목(서브태스크)을 추가할 수 있습니다.
- 서브태스크 완료율이 상위 항목에 표시됩니다.

### D-Day 순 정렬

할 일을 마감일(D-Day) 기준으로 정렬하여 볼 수 있습니다.

- 할 일 목록 상단의 정렬 옵션에서 **D-Day 순**을 선택합니다.
- 마감일이 가까운 순서대로 플랫 리스트로 표시됩니다.
- 그룹 구분 없이 전체 할 일을 마감일 기준으로 한눈에 확인할 수 있습니다.

### 시간 설정 및 타임라인 통합 뷰

할 일에 시간을 지정하여 시간순으로 관리할 수 있습니다.

**시간 설정 방법:**
1. 할 일 추가 폼에서 ⏰ 시간 입력란에 시간을 입력합니다 (선택사항).
2. 시간이 설정된 할 일은 시간순으로 자동 정렬됩니다.
3. 기존 할 일도 더블클릭하여 시간을 추가할 수 있습니다.

**통합 보기 (타임라인):**
할 일 페이지 상단의 "통합 보기"에서 수업·일정을 할 일과 함께 볼 수 있습니다.

1. **📚 수업**: 오늘의 시간표 교시가 시간순으로 할 일 목록에 표시됩니다.
2. **📅 일정**: 오늘의 일정이 시간순으로 할 일 목록에 표시됩니다.

시간표와 일정은 읽기 전용으로 표시되며, 체크하거나 삭제할 수 없습니다.
대시보드의 "오늘 할 일" 위젯에서도 동일한 통합 보기가 적용됩니다.

### D-Day 설정

- 중요 일정(시험, 행사 등)에 D-Day를 설정하면 대시보드에 남은 일수가 표시됩니다.
- 여러 개의 D-Day를 동시에 설정할 수 있습니다.

## 일정 관리

### 학급 행사 등록

1. 일정 관리 페이지에서 날짜를 클릭합니다.
2. 일정 제목, 시간, 카테고리를 입력합니다.
3. 컬러 라벨을 선택하여 카테고리별로 구분할 수 있습니다.

### 캘린더 뷰

- 월간/주간/일간 뷰를 전환할 수 있습니다.
- 일정이 있는 날짜에 점 표시가 되어 한눈에 파악할 수 있습니다.

### 같은 날짜 일정 수동 정렬

- 같은 날짜에 여러 일정이 있을 때, 드래그앤드롭으로 일정의 표시 순서를 자유롭게 변경할 수 있습니다.
- 일정 목록에서 일정을 길게 누르고 원하는 위치로 끌어다 놓으면 순서가 저장됩니다.

### 일정 검색

- 키워드로 전체 일정을 검색할 수 있습니다.
- 연도 필터를 적용하여 특정 연도의 일정만 조회할 수 있습니다.

### 일정 일괄 삭제

- 일정 관리 페이지에서 여러 일정을 한 번에 삭제할 수 있습니다.
- **다중 선택 삭제**: 체크박스로 여러 일정을 선택하여 한 번에 삭제
- **카테고리별 삭제**: 특정 카테고리의 모든 일정을 일괄 삭제
- **기간별 삭제**: 시작일~종료일 범위의 일정을 일괄 삭제

### 반복 일정 예외 날짜

반복 일정에서 특정 날짜를 제외할 수 있습니다.

1. 반복 일정을 편집합니다.
2. **"예외 날짜"** 항목에서 제외할 날짜를 추가합니다.
3. 공휴일, 행사일 등 반복 일정이 없는 날을 간편하게 관리할 수 있습니다.

> **예시**: 매주 월요일 교직원 회의가 있지만, 개학식 날은 제외하고 싶을 때 예외 날짜를 추가하세요.

### 일정 카테고리 관리

- 일정 카테고리의 이름을 변경할 수 있습니다.
- 기본 카테고리의 이름도 직접 수정할 수 있습니다.
- 삭제 버튼이 더 잘 보이도록 개선되었습니다.

### D-Day 카운트다운 위젯

- 일정에 D-Day를 설정하면 대시보드와 위젯 모드에서 남은 일수가 표시됩니다.

### .ssampin 파일 공유

- 작성한 일정을 `.ssampin` 파일로 내보내 동료 교사와 공유할 수 있습니다.
- 받은 `.ssampin` 파일을 더블클릭하면 쌤핀 앱이 자동으로 실행되며 가져오기 화면이 표시됩니다.
- 가져오기 시 중복 일정은 자동으로 감지하여 알려줍니다.

### Google 캘린더 연동

- 설정에서 Google 캘린더를 연동하면 Google 캘린더의 일정이 쌤핀에 함께 표시됩니다.
- 연동 및 해지는 설정에서 언제든지 변경할 수 있습니다.
- 현재 읽기 전용으로, 쌤핀에서 Google 캘린더로의 일정 추가는 추후 지원 예정입니다.

### NEIS 학사일정 연동

- NEIS에서 제공하는 학사일정(개학, 방학, 공휴일 등)을 자동으로 가져와 일정 캘린더에 표시합니다.

### 일정 등록 — 상세 가이드

새로운 일정을 등록하는 전체 과정입니다.

1. 사이드바에서 **"일정 관리"** 를 클릭합니다.
2. 캘린더에서 **원하는 날짜를 클릭**합니다.
3. 일정 등록 폼이 나타나면 **제목**을 입력합니다.
4. **카테고리**를 선택합니다 (예: 학교행사, 학급행사, 개인일정 등).
5. 필요하면 **시작 시간**과 **종료 시간**을 설정합니다.
6. 필요하면 **교시**를 지정할 수 있습니다 (예: 3교시, 5~6교시).
7. **컬러 라벨**을 선택하여 시각적으로 구분합니다.
7. **"저장"** 버튼을 클릭하면 캘린더에 일정이 표시됩니다.

**일정 수정**: 캘린더에서 일정을 클릭하면 편집할 수 있습니다.
**일정 삭제**: 편집 화면에서 삭제 버튼을 클릭합니다.

### 캘린더 뷰 전환 방법

월간, 주간, 일간 뷰를 전환하여 일정을 확인합니다.

1. 일정 관리 페이지 상단에서 **뷰 전환 버튼**을 찾습니다.
2. **"월"** 을 클릭하면 한 달의 일정을 한눈에 볼 수 있습니다.
3. **"주"** 를 클릭하면 주간 일정을 시간대별로 확인합니다.
4. **"일"** 을 클릭하면 하루의 일정을 상세하게 봅니다.
5. **좌우 화살표**로 이전/다음 월, 주, 일로 이동합니다.
6. **"오늘"** 버튼으로 오늘 날짜로 빠르게 돌아갑니다.

### D-Day 설정 방법

중요한 일정까지 남은 일수를 대시보드에 표시합니다.

1. 일정을 등록하거나 편집할 때 **"D-Day"** 옵션을 켭니다.
2. D-Day가 설정된 일정은 대시보드의 **D-Day 위젯**에 남은 일수와 함께 표시됩니다.
3. 위젯 모드에서도 D-Day를 확인할 수 있습니다.
4. D-Day를 여러 개 설정하면 모두 대시보드에 표시됩니다.

> **활용 팁**: 중간고사, 기말고사, 수학여행 등 중요한 학교 일정에 D-Day를 설정하면 학생들과 함께 카운트다운을 할 수 있어요.

### 일정 카테고리 관리 — 상세 가이드

일정 카테고리를 추가, 수정, 삭제하는 방법입니다.

1. 일정 관리 페이지에서 **카테고리 관리 아이콘**을 클릭합니다.
2. **새 카테고리 추가**: 카테고리명을 입력하고 색상을 선택합니다.
3. **카테고리 이름 변경**: 기존 카테고리를 클릭하여 이름을 수정합니다.
4. **카테고리 삭제**: 삭제 버튼을 클릭합니다. 해당 카테고리의 일정은 삭제되지 않습니다.

### NEIS 학사일정 자동 연동 — 상세 가이드

NEIS에서 학사일정을 자동으로 가져오는 방법입니다.

1. **설정 → 학교 정보 탭**에서 학교가 등록되어 있는지 확인합니다.
2. **설정 → 일정 탭**에서 NEIS 학사일정 가져오기가 활성화되어 있는지 확인합니다.
3. 설정이 완료되면 **개학, 방학, 공휴일, 학교 행사** 등이 자동으로 캘린더에 표시됩니다.
4. NEIS에서 가져온 일정은 별도 표시가 되어 직접 등록한 일정과 구분됩니다.

**NEIS 학사일정이 안 나올 때:**
- 학교 정보가 올바르게 등록되어 있는지 확인하세요.
- 인터넷 연결 상태를 확인하세요.
- NEIS API 점검 시간(새벽)에는 일시적으로 가져오지 못할 수 있어요.

### Google 캘린더 연동 — 상세 가이드

Google 캘린더의 일정을 쌤핀에서 함께 확인하는 방법입니다.

1. **사이드바 → 설정 → 동기화 탭**으로 이동합니다.
2. **"Google 캘린더" 섹션**에서 **"연결"** 버튼을 클릭합니다.
3. Google 로그인 화면에서 **계정을 선택**합니다.
4. **캘린더 접근 권한**을 허용합니다.
5. 연결이 완료되면 Google 캘린더의 일정이 쌤핀 캘린더에 함께 표시됩니다.
6. Google 캘린더 일정은 아이콘으로 구분되어 표시됩니다.

**연결 해제 방법:**
1. 설정 → 동기화 탭에서 Google 캘린더 섹션의 **"연결 해제"** 버튼을 클릭합니다.
2. 해제 후 Google 캘린더 일정은 더 이상 표시되지 않습니다.

> **참고**: Google 캘린더 연동은 현재 읽기 전용입니다. Google 캘린더의 일정을 보기만 할 수 있으며, 쌤핀에서 Google 캘린더로 일정을 추가하는 기능은 추후 지원 예정입니다.

### 일정 검색 및 필터 — 상세 가이드

등록된 일정을 키워드와 필터로 빠르게 찾는 방법입니다.

1. 일정 관리 페이지에서 **검색 아이콘(🔍)** 을 클릭합니다.
2. 검색란에 **키워드**를 입력합니다 (예: "시험", "체육대회").
3. 일치하는 일정이 목록으로 표시됩니다.
4. **연도 필터**를 적용하면 특정 연도의 일정만 조회할 수 있습니다.
5. 검색 결과에서 일정을 클릭하면 해당 날짜로 이동합니다.

### 일정 일괄 삭제 — 상세 가이드

여러 일정을 한 번에 정리하는 방법입니다.

1. 일정 관리 페이지에서 **일괄 삭제 버튼**을 클릭합니다.
2. 삭제 방법을 선택합니다:
   - **다중 선택**: 개별 일정에 체크 표시하여 선택
   - **카테고리별**: 특정 카테고리의 모든 일정 선택
   - **기간별**: 시작일과 종료일을 지정하여 해당 기간의 일정 선택
3. 선택이 완료되면 **"삭제"** 버튼을 클릭합니다.
4. 확인 팝업에서 **"확인"** 을 클릭하면 일정이 삭제됩니다.

> **주의**: 삭제된 일정은 복구할 수 없어요. 중요한 일정은 먼저 백업해 두세요.

## 쌤핀 모바일

쌤핀 모바일은 스마트폰에서 시간표, 출결, 메모, 할일, 일정을 확인하고 관리할 수 있는 모바일 웹앱(PWA)입니다.

### 시작하기

1. **접속**: 스마트폰 브라우저에서 [m.ssampin.com](https://m.ssampin.com)에 접속합니다.
2. **홈 화면 추가**: 앱처럼 사용하려면 홈 화면에 추가하세요.
   - **iOS**: Safari에서 공유 버튼 → "홈 화면에 추가"
   - **Android**: 설치 배너가 표시되거나, 메뉴(⋮) → "홈 화면에 추가"
3. **Google Drive 연동**: PC 쌤핀에서 먼저 Google Drive 동기화를 설정한 후, 모바일에서 같은 Google 계정으로 로그인하면 데이터가 자동으로 동기화됩니다.

### 주요 기능

- **시간표 확인**: PC에서 설정한 시간표를 모바일에서 바로 확인
- **출결 관리**: 교실에서 스마트폰으로 바로 출결 체크 (출석·결석·지각·조퇴 터치로 기록)
- **메모 확인**: PC에서 작성한 메모를 모바일에서 확인
- **할일 확인**: 오늘의 할일 목록을 모바일에서 확인
- **일정 확인**: 등록된 일정을 모바일에서 확인
- **쌤도구**: 타이머, 랜덤뽑기, 점수판, 소음측정기 등 PC의 쌤도구 9종을 모바일에서도 바로 사용할 수 있습니다.

### Google Drive 동기화

- PC 쌤핀에서 작성한 모든 데이터는 Google Drive를 통해 모바일과 자동 동기화됩니다.
- 별도 서버 없이 Google Drive만으로 안전하게 연동됩니다.
- PC에서 데이터를 변경하면 모바일에서 새로고침 시 반영됩니다.

### 주의사항

- 모바일 앱은 인터넷 연결이 필요합니다.
- PC 쌤핀에서 Google Drive 동기화를 먼저 설정해야 합니다.
- 시간표·좌석배치 등 편집은 PC에서 수행합니다. 모바일에서는 쌤도구와 출결을 포함한 주요 기능을 사용할 수 있습니다.

## 동기화 완전 가이드

### Google Drive 동기화 — 처음 설정하기

PC 쌤핀에서 Google Drive 동기화를 처음 설정하는 전체 과정입니다.

1. 사이드바에서 **"설정"** 을 클릭합니다.
2. **"동기화" 탭**을 클릭합니다.
3. **"동기화 사용"** 토글을 켭니다.
4. **"Google 계정 연결"** 버튼을 클릭합니다.
5. 브라우저에서 Google 로그인 화면이 열립니다. **Google 계정을 선택하고 로그인**합니다.
6. "쌤핀이 Google Drive에 액세스하도록 허용하시겠습니까?" 화면에서 **"허용"** 을 클릭합니다.
7. 쌤핀으로 돌아오면 연결된 **이메일 주소**가 표시됩니다.
8. **"클라우드로 업로드"** 버튼을 클릭하여 현재 데이터를 처음으로 업로드합니다.

> **중요**: Google Drive의 "쌤핀 동기화" 폴더에 데이터가 저장됩니다. 이 폴더를 직접 삭제하면 동기화 데이터가 유실될 수 있으니 주의하세요.

### Google Drive 동기화 — 수동 백업 및 복원

수동으로 데이터를 클라우드에 업로드하거나 다운로드하는 방법입니다.

1. **사이드바 → 설정 → 동기화 탭**으로 이동합니다.
2. **"클라우드로 업로드"** 버튼: 현재 PC의 데이터를 Google Drive에 업로드합니다.
3. **"클라우드에서 다운로드"** 버튼: Google Drive의 데이터를 PC로 가져옵니다.
4. 동기화 결과가 요약으로 표시됩니다.

**데이터 충돌 시:** 가장 최근에 수정된 데이터가 우선 적용됩니다(latest 정책).

### Google Drive 동기화 — 자동 동기화 설정

자동으로 데이터가 동기화되도록 설정하는 방법입니다.

1. **동기화 탭**에서 Google 계정이 연결되어 있는지 확인합니다.
2. **"시작 시 자동 동기화"**: 앱을 시작할 때 자동으로 클라우드 데이터를 확인합니다.
3. **"저장 시 자동 동기화"**: 데이터를 저장할 때마다 자동으로 업로드합니다.
4. **"자동 동기화 간격"**: 5분/10분/15분/30분 단위로 주기적 동기화를 설정합니다.
5. 원하는 옵션을 선택하고 **"저장"** 을 클릭합니다.

> **추천 설정**: "시작 시 자동 동기화" + "저장 시 자동 동기화"를 모두 켜면 데이터 유실 걱정이 없어요.

### PC와 모바일 동기화 설정

PC 쌤핀의 데이터를 모바일(스마트폰)에서 확인하는 방법입니다.

1. **PC 쌤핀**에서 설정 → 동기화 탭에서 Google Drive 동기화를 설정합니다 (위의 "처음 설정하기" 참조).
2. PC에서 **"클라우드로 업로드"** 를 한 번 실행하여 데이터를 업로드합니다.
3. **스마트폰 브라우저**에서 [m.ssampin.com](https://m.ssampin.com)에 접속합니다.
4. 모바일 앱에서 **"Google 로그인"** 을 클릭합니다.
5. **PC와 동일한 Google 계정**으로 로그인합니다.
6. 자동으로 데이터가 동기화되어 시간표, 메모, 할일, 일정을 확인할 수 있습니다.

**주의사항:**
- 반드시 PC에서 먼저 Google Drive 동기화를 설정해야 합니다.
- PC와 모바일은 같은 Google 계정을 사용해야 합니다.
- 모바일에서 변경한 출결 데이터는 PC에서 동기화 시 반영됩니다.

### 동기화 문제 해결

동기화가 정상적으로 작동하지 않을 때 확인할 사항입니다.

1. **인터넷 연결**이 되어 있는지 확인하세요.
2. **Google 계정 연결 상태**를 확인하세요. 토큰이 만료되면 재연결이 필요합니다.
3. **재연결 방법**: 설정 → 동기화 탭에서 "연결 해제" → 다시 "Google 계정 연결"을 클릭합니다.
4. **클라우드 데이터 확인**: Google Drive에서 "쌤핀 동기화" 폴더가 있는지 확인합니다.
5. **수동 동기화 시도**: "클라우드로 업로드" 또는 "클라우드에서 다운로드" 버튼으로 직접 동기화해 보세요.

**Google 로그인이 안 될 때:**
- 카카오톡, 네이버 등 인앱 브라우저에서는 Google 보안 정책으로 로그인이 차단돼요.
- Chrome, Safari 등 기본 브라우저에서 접속하세요.

### 클라우드 데이터 삭제

Google Drive에 저장된 동기화 데이터를 삭제하는 방법입니다.

1. **설정 → 동기화 탭**으로 이동합니다.
2. 하단의 **"클라우드 데이터 삭제"** 버튼을 클릭합니다.
3. 확인 팝업에서 **"삭제"** 를 클릭합니다.
4. Google Drive의 쌤핀 동기화 폴더가 삭제됩니다.

> **주의**: 클라우드 데이터를 삭제해도 PC의 로컬 데이터는 유지됩니다. 하지만 모바일에서는 데이터를 확인할 수 없게 됩니다.

### 데이터 수동 백업 (.ssampin 파일)

Google Drive를 사용하지 않고 데이터를 백업하는 방법입니다.

1. 사이드바에서 **"설정"** 을 클릭합니다.
2. **"앱 정보" 탭** 또는 **데이터 관리 섹션**에서 **"백업"** 버튼을 클릭합니다.
3. 파일 저장 대화상자에서 원하는 위치를 선택하고 저장합니다.
4. `.ssampin` 확장자의 백업 파일이 생성됩니다.
5. 이 파일을 USB, 이메일, 클라우드 등으로 안전하게 보관하세요.

**복원 방법:**
1. 설정에서 **"복원"** 버튼을 클릭합니다.
2. 이전에 저장한 `.ssampin` 백업 파일을 선택합니다.
3. 데이터가 복원됩니다.

> **팁**: 학기 초와 학기 말에 백업 파일을 만들어 두면 만일의 사태에 대비할 수 있어요.

### 학교 정보 탭 — 상세 가이드

학교를 등록하고 NEIS 연동을 설정하는 탭입니다. 경로: **사이드바 → 설정 → 학교 정보 탭**

1. **학교 검색**: 학교 이름을 입력하고 "검색" 버튼을 클릭합니다.
2. **학교 선택**: 검색 결과에서 내 학교를 클릭합니다. 같은 이름의 학교가 여러 개인 경우 주소로 구분하세요.
3. **학년/반 입력**: 담당 학년과 반 번호를 입력합니다.
4. **급식용 학교 설정**: 급식을 제공하는 학교가 별도인 경우(급식공동조리학교 등) 별도로 설정할 수 있습니다.
5. **저장**: "저장" 버튼을 클릭하면 NEIS에서 시간표와 급식 정보를 자동으로 가져옵니다.

> **주의**: 학교 정보가 정확하지 않으면 시간표, 급식, 학사일정이 모두 정상적으로 표시되지 않을 수 있어요.

### 교시 시간 탭 — 상세 가이드

교시별 시작/종료 시간을 설정하는 탭입니다. 경로: **사이드바 → 설정 → 교시 시간 탭**

1. 학교급(초/중/고)에 맞는 **기본 프리셋**이 자동 적용됩니다.
2. 각 교시의 **시작 시간**과 **종료 시간**을 직접 수정할 수 있습니다.
3. 점심시간 교시를 직접 선택하여 설정할 수 있습니다. (학교마다 다른 점심시간을 정확하게 반영)
4. 교시 수를 늘리거나 줄일 수 있습니다.
5. **NEIS 시간표 자동 동기화** 설정이 이 탭에서 관리됩니다.
6. "저장"을 클릭하면 대시보드의 "현재 교시" 위젯에 즉시 반영됩니다.

> **활용 팁**: 단축수업일이나 시험 기간에는 교시 시간을 임시로 변경하면 대시보드에서 정확한 현재 교시를 확인할 수 있어요.

### NEIS 학사일정 연동 설정

NEIS에서 학사일정을 가져와 일정 캘린더에 표시하는 방법입니다. 경로: **사이드바 → 설정 → 일정 탭**

1. **일정 탭**을 클릭합니다.
2. **"NEIS 학사일정 가져오기"** 옵션을 확인합니다.
3. 학교가 등록되어 있으면 자동으로 학사일정(개학, 방학, 공휴일 등)을 가져옵니다.
4. 가져온 학사일정은 일정 관리 캘린더에 별도 카테고리로 표시됩니다.

### 날씨 지역 설정 방법

대시보드에 표시되는 날씨 정보의 지역을 설정합니다. 경로: **사이드바 → 설정 → 날씨 탭**

1. **날씨 탭**을 클릭합니다.
2. **드롭다운**에서 지역을 선택하거나, **검색란**에 도시명을 입력합니다.
3. 전국 83개 시/군 지역을 지원합니다.
4. 지역을 선택하면 대시보드 날씨 위젯에 즉시 반영됩니다.

> **참고**: 날씨 정보는 인터넷 연결이 필요합니다. 오프라인 상태에서는 마지막으로 가져온 날씨 정보가 표시됩니다.

### PIN 잠금 설정 방법

민감한 메뉴에 비밀번호를 설정하여 학생의 접근을 차단합니다. 경로: **사이드바 → 설정 → 보안 탭**

1. **보안 탭**을 클릭합니다.
2. **"PIN 잠금 사용"** 토글을 켭니다.
3. **4자리 숫자 PIN**을 입력합니다.
4. **잠금 대상 메뉴**를 개별적으로 선택합니다 (담임 업무, 수업 관리, 즐겨찾기 등).
5. "저장"을 클릭합니다.

설정 후 잠금 대상 메뉴에 접근하면 PIN 입력 화면이 나타납니다. 올바른 PIN을 입력해야 메뉴를 사용할 수 있습니다.

**PIN을 잊었을 때**: 앱을 완전히 종료하고 재시작한 후, 설정에서 PIN을 재설정하세요.

### 테마 변경 방법

앱의 전체적인 색상 테마를 변경합니다. 경로: **사이드바 → 설정 → 디스플레이 탭**

1. **디스플레이 탭**을 클릭합니다.
2. **11가지 테마 프리셋** 중 원하는 테마를 클릭합니다.
   - 기본 7가지 외에 노션 라이트/다크, 크래프트 노트 라이트/다크 프리셋이 추가되었습니다.
3. 미리보기가 즉시 표시됩니다.
4. **커스텀 테마**를 만들려면 각 색상 항목을 직접 수정합니다.
5. "저장"을 클릭하면 앱 전체에 테마가 적용됩니다.

### 폰트 변경 방법

앱 전체에 적용되는 한글 폰트를 변경합니다. 경로: **사이드바 → 설정 → 디스플레이 탭**

1. **디스플레이 탭**에서 **"글꼴" 섹션**을 찾습니다.
2. **11종의 한글 폰트** 중 원하는 폰트를 선택합니다.
3. 미리보기에서 폰트 스타일을 확인합니다.
4. "저장"을 클릭하면 앱 전체 텍스트에 선택한 폰트가 적용됩니다.

### 내 폰트 업로드 (커스텀 폰트)

직접 다운로드한 폰트 파일을 업로드하여 앱 전체에 적용할 수 있습니다.

1. **설정 → 디스플레이 탭 → 글꼴** 섹션 하단의 **"내 폰트"** 영역을 찾습니다.
2. **"폰트 파일 업로드"** 버튼을 클릭합니다.
3. .woff2, .woff, .ttf, .otf 형식의 폰트 파일을 선택합니다 (10MB 이하).
4. 폰트 이름을 입력하면 즉시 적용됩니다.
5. 업로드한 폰트는 "내 폰트" 항목에 표시되며, 언제든 선택하거나 삭제할 수 있습니다.
6. 폰트를 삭제하면 기본 폰트(Noto Sans KR)로 자동 복귀됩니다.

> **팁**: 눈누(noonnu.cc) 등 무료 한글 폰트 사이트에서 좋아하는 폰트를 다운받아 적용해보세요!

### 사이드바 메뉴 커스터마이즈

사이드바에 표시되는 메뉴의 순서와 표시 여부를 변경합니다. 경로: **사이드바 → 설정 → 사이드바 탭**

1. **사이드바 탭**을 클릭합니다.
2. 각 메뉴 항목의 **토글**을 켜거나 꺼서 표시/숨김을 설정합니다.
3. 메뉴 항목을 **드래그**하여 순서를 변경합니다.
4. "저장"을 클릭하면 사이드바에 즉시 반영됩니다.

> **팁**: 자주 쓰지 않는 메뉴(예: 메모)를 숨기면 사이드바가 깔끔해져요.

### Google Drive 동기화 설정

Google Drive를 통해 데이터를 백업하고 여러 기기에서 동기화합니다. 경로: **사이드바 → 설정 → 동기화 탭**

1. **동기화 탭**을 클릭합니다.
2. **"동기화 사용"** 토글을 켭니다.
3. **"Google 계정 연결"** 버튼을 클릭합니다.
4. Google 로그인 화면에서 **계정을 선택하고 로그인**합니다.
5. 권한 요청을 **"허용"** 합니다.
6. 연결이 완료되면 이메일이 표시됩니다.

**동기화 옵션:**
- **시작 시 자동 동기화**: 앱 시작 시 자동으로 클라우드에서 데이터를 가져옵니다.
- **저장 시 자동 동기화**: 데이터가 변경될 때마다 자동으로 클라우드에 업로드합니다.
- **자동 동기화 간격**: 5분/10분/15분/30분 단위로 자동 동기화 주기를 설정합니다.
- **수동 동기화**: "클라우드로 업로드" 또는 "클라우드에서 다운로드" 버튼으로 수동 동기화합니다.

### 시스템 설정 방법

시스템 관련 설정을 변경합니다. 경로: **사이드바 → 설정 → 시스템 탭**

1. **시스템 탭**을 클릭합니다.
2. **"시작 시 자동 실행"**: 컴퓨터 부팅 시 쌤핀이 자동으로 실행됩니다.
3. **"창 닫기 동작"**: 창을 닫을 때 동작을 선택합니다 — **위젯 모드로 전환** / **트레이로 최소화** / **매번 물어보기** 중 선택할 수 있습니다. 설정 → 위젯 탭에서도 변경할 수 있습니다.
4. **"알림 소리"**: 일정 알림, 타이머 종료 등의 알림 소리를 켜거나 끕니다.
4. **"방해 금지 시간"**: 지정된 시간대에는 모든 알림을 끕니다. 수업 시간에 유용합니다.

### Google 캘린더 연동 설정

Google 캘린더의 일정을 쌤핀에 함께 표시합니다. 경로: **사이드바 → 설정 → 동기화 탭**

1. **동기화 탭**에서 **"Google 캘린더" 섹션**을 찾습니다.
2. **"Google 캘린더 연결"** 버튼을 클릭합니다.
3. Google 로그인 후 캘린더 접근 권한을 허용합니다.
4. 연결이 완료되면 Google 캘린더의 일정이 쌤핀 일정 관리에 함께 표시됩니다.
5. 연결을 해제하려면 **"연결 해제"** 버튼을 클릭합니다.

> **참고**: 현재 Google 캘린더 연동은 읽기 전용입니다. Google 캘린더에서 쌤핀으로 일정을 가져오기만 하며, 쌤핀에서 Google 캘린더로 일정을 추가하는 기능은 추후 지원 예정입니다.

### 앱 정보 및 업데이트 확인

현재 버전 정보를 확인하고 업데이트합니다. 경로: **사이드바 → 설정 → 앱 정보 탭**

1. **앱 정보 탭**을 클릭합니다.
2. 현재 설치된 **버전 번호**와 **릴리스 날짜**를 확인할 수 있습니다.
3. **"업데이트 확인"** 버튼으로 새 버전이 있는지 확인합니다.
4. 새 버전이 있으면 **"업데이트"** 버튼을 클릭하면 자동으로 다운로드 및 설치됩니다.
5. **릴리스 노트**에서 각 버전의 변경 사항을 확인할 수 있습니다.

### PIN 잠금

- 4자리 PIN을 설정하여 민감한 메뉴(담임 업무, 수업 관리, 즐겨찾기 등)에 접근을 제한할 수 있습니다.

- 기능별로 잠금 대상을 개별 설정할 수 있습니다.

### 데이터 저장 위치

- 모든 데이터는 내 컴퓨터에만 저장됩니다. 서버로 전송되지 않습니다.
- Electron 앱: `%APPDATA%/ssampin/data/` 폴더에 JSON 파일로 저장
- 저장 위치는 설정에서 확인할 수 있습니다.

### 백업 및 복원

- 설정에서 데이터를 백업 파일로 내보내 안전하게 보관할 수 있습니다.
- 백업 파일을 불러와 데이터를 복원할 수 있습니다.
- 컴퓨터를 교체할 때 데이터를 이전하는 데 사용합니다.

### Google Drive 동기화

- Google Drive 계정을 연동하면 쌤핀 데이터를 클라우드에 백업하고 복원할 수 있습니다.
- 여러 컴퓨터에서 동일한 데이터를 유지하거나, 데이터 손실을 방지하는 데 활용합니다.
- 설정에서 연동 및 해지를 언제든지 변경할 수 있습니다.

### 테마 설정

- 7가지 테마 프리셋 중 원하는 테마를 선택할 수 있습니다.
- 커스텀 테마를 직접 만들어 색상을 자유롭게 변경할 수 있습니다.

### 폰트 설정

- 11종의 한글 폰트 중 원하는 폰트를 선택할 수 있습니다.
- 직접 다운로드한 폰트 파일(.woff2, .ttf, .otf)을 업로드하여 사용할 수도 있습니다.
- 선택한 폰트가 앱 전체에 적용됩니다.

### 사이드바 커스터마이즈

- 사이드바 메뉴의 순서를 변경하거나, 자주 사용하지 않는 메뉴를 숨길 수 있습니다.

### 행사 알림 팝업

- 등록한 일정의 당일 또는 D-Day 알림 팝업을 켜거나 끌 수 있습니다.

### 앱 업데이트

- 앱이 자동으로 새 버전을 확인합니다.
- 새 버전이 있으면 알림이 표시되며, "업데이트" 버튼만 누르면 자동으로 업데이트됩니다.
- 업데이트 시 릴리스 노트 모달이 표시되어 어떤 기능이 추가/수정되었는지 확인할 수 있습니다.
- 수동으로 업데이트하려면 웹사이트에서 최신 설치 파일을 다운로드하여 설치합니다.

## 급식 정보

### NEIS API 연동

- NEIS(나이스) 공식 API에서 급식 정보를 자동으로 가져옵니다.
- 설정에서 학교만 검색하여 등록하면 매일 급식 메뉴가 대시보드에 표시됩니다.

### 표시 내용

- 오늘의 점심 메뉴가 대시보드 위젯에 표시됩니다.
- 알레르기 정보가 함께 표시됩니다.

### 주의사항

- 급식 정보는 인터넷 연결이 필요합니다.
- NEIS API 점검 시간에는 일시적으로 표시되지 않을 수 있습니다.
- 학교가 등록되지 않은 경우 급식 정보가 표시되지 않습니다.

### 급식 메뉴 직접 입력

NEIS에 급식 정보가 없거나 직접 메뉴를 관리하고 싶을 때:

1. 급식 페이지에서 **수기 입력** 모드를 선택합니다.
2. 날짜를 선택하고 메뉴를 직접 입력합니다.
3. **CSV 파일 불러오기**: CSV 형식의 급식 메뉴 파일을 업로드하여 한 번에 등록할 수 있습니다.

> **팁**: NEIS 급식 정보와 수기 입력을 병행할 수 있어요. 방학 중 급식이나 특별 메뉴를 직접 입력해두면 유용해요.

## 자주 묻는 질문 (FAQ)

### 무료인가요?
네, 완전 무료예요. 광고도 없어요.

### 인터넷 없이도 되나요?
날씨, 급식, Google 캘린더/Drive 연동, 공유 기능을 제외한 모든 기능이 오프라인에서 동작해요.

### Mac에서도 쓸 수 있나요?
네, macOS도 지원해요! ssampin.com에서 macOS용 DMG 파일을 다운로드하세요. Apple Silicon(M1~M4)과 Intel Mac 모두 지원합니다. 처음 실행 시 "개발자를 확인할 수 없음" 경고가 뜰 수 있어요. 시스템 설정 → 개인정보 보호 및 보안에서 "확인 없이 열기"를 클릭하거나, 앱을 Control+클릭 → "열기"를 선택하세요.

### 데이터는 어디에 저장되나요?
내 컴퓨터에만 저장돼요. 서버로 전송되지 않아요. Google Drive 동기화를 설정하면 클라우드에 백업할 수 있어요.

### 업데이트는 어떻게 하나요?
앱이 자동으로 새 버전을 알려줘요. 알림이 오면 "업데이트" 버튼만 누르면 돼요.

### 바이러스 아닌가요? 경고가 떠요!
개인 개발 앱이라 Microsoft 인증서가 아직 없어요. 경고 종류에 따라 해결 방법이 달라요. "Microsoft Windows의 PC 보호" 화면이면 "추가 정보 → 실행"을 클릭하세요. "스마트 앱 컨트롤이 차단했습니다" 화면(Windows 11)이면 설치 파일 우클릭 → 속성 → "차단 해제" 체크 → 확인 후 다시 실행하세요.

### 급식 정보는 어떻게 나오나요?
NEIS(나이스) 공식 API에서 자동으로 가져와요. 설정에서 학교만 검색하면 매일 급식이 표시돼요.

### 쌤도구는 뭔가요?
타이머, 랜덤 뽑기, 투표, 주관식 설문, 워드클라우드, 점수판, 룰렛, QR코드 등 수업에 바로 쓸 수 있는 15가지 교실 도구예요.

### PIN 잠금은 왜 필요한가요?
학생이 볼 수 있는 상황에서 담임 업무나 수업 관리 같은 민감한 정보를 보호할 수 있어요. 기능별로 잠금을 설정할 수 있어요.

### 과제수합은 어떻게 사용하나요?
수업 관리 페이지에서 과제를 생성한 후, QR코드나 URL을 학생들에게 공유하면 스마트폰으로 바로 제출할 수 있어요. 파일과 텍스트 제출을 모두 지원해요. Google Drive 연동 시 파일이 클라우드에 저장돼요.

### 과목 색상을 바꿀 수 있나요?
네! 시간표에서 과목을 클릭하면 인라인 팔레트가 나타나요. 16가지 프리셋 색상 중 원하는 색을 선택하거나 기본값으로 되돌릴 수 있어요.

### 짝꿍 배치는 어떻게 하나요?
좌석배치 페이지에서 "짝꿍 모드"를 활성화하면 2인 1조 짝꿍 배치를 자동으로 생성할 수 있어요. 3인짝 모드도 지원해요. 내보내기에도 반영돼요.

### 담임 메모(학생 기록)를 내보낼 수 있나요?
네! 담임 업무 → 학생 기록 탭에서 기간/카테고리/학생 필터를 적용한 후 Excel, HWPX(한글), 생활기록부용 Excel 형식으로 내보낼 수 있어요.

### 담임메모 입력 시 이전 기록을 참고할 수 있나요?
네! v1.8.0부터 입력 화면이 3컬럼 레이아웃으로 개편되어, 오른쪽 참조 패널에서 "오늘" 탭과 "이력" 탭으로 이전 기록을 보면서 새 기록을 입력할 수 있어요. 이력 탭에서는 과거 기록을 펼쳐보고 인라인으로 수정할 수도 있어요.

### 담임메모에서 나이스 미반영 기록을 확인할 수 있나요?
네! 진행현황 탭에서 나이스 미반영 출결 기록을 학생별로 드릴다운하여 확인할 수 있어요. 검색 탭의 액션 대시보드에서도 나이스 미반영 건수를 클릭하면 바로 해당 기록을 필터링할 수 있어요.

### 상담 예약은 어떻게 만드나요?
담임 업무 → 상담 예약 탭에서 위자드 형태의 단계별 안내에 따라 상담 일정을 설정하세요. 수업 시간은 자동으로 차단되고, 생성된 QR코드나 URL을 학생/학부모와 공유하면 돼요.

### 위젯 모드에서 메모를 보려면 어떻게 하나요?
위젯 설정에서 "메모 포커스" 위젯을 추가하세요. 이전/다음 버튼으로 여러 메모를 순서대로 확인할 수 있어요.

### Google Drive 동기화는 어떻게 설정하나요?
설정 → Google Drive 동기화 항목에서 Google 계정으로 연동하면 돼요. 연동 후 수동 또는 자동으로 데이터를 백업/복원할 수 있어요.

### 일정을 검색할 수 있나요?
네! 일정 관리 페이지에서 검색 기능을 사용하면 키워드로 전체 일정을 찾을 수 있어요. 연도 필터도 지원해요.

### 모바일에서도 쌤핀을 쓸 수 있나요?
네! m.ssampin.com에 접속하면 스마트폰에서도 시간표, 출결, 메모, 할일, 일정을 확인할 수 있어요. 홈 화면에 추가하면 앱처럼 사용 가능해요. PC에서 Google Drive 동기화를 먼저 설정해야 해요.

### 모바일에서 출결 체크가 되나요?
네! 모바일 앱에서 출석·결석·지각·조퇴를 터치로 바로 기록할 수 있어요. 기록은 Google Drive를 통해 PC와 자동 동기화돼요.

### 모바일과 PC 데이터는 어떻게 연동하나요?
Google Drive를 통해 자동 동기화돼요. PC 쌤핀에서 설정 → Google Drive 동기화를 먼저 설정하고, 모바일에서 같은 Google 계정으로 로그인하면 데이터가 자동으로 연동돼요.

### 일정을 한꺼번에 삭제할 수 있나요?
네! 일정 관리 페이지에서 일괄 삭제 기능을 사용하면 돼요. 여러 일정을 체크해서 한 번에 삭제하거나, 카테고리별·기간별로 한꺼번에 정리할 수 있어요.

### 자주 쓰는 쌤도구를 바로 실행할 수 있나요?
네! 대시보드의 '자주 쓰는 도구' 위젯에서 타이머, 랜덤 뽑기, 점수판 등 원하는 도구를 최대 8개까지 등록하고 바로 실행할 수 있어요. 편집 버튼으로 도구를 자유롭게 변경할 수 있어요.

### 카카오톡에서 링크를 열면 Google 로그인이 안 돼요
카카오톡, 네이버 등 인앱 브라우저에서는 Google 보안 정책으로 로그인이 차단돼요. 링크를 길게 눌러 '외부 브라우저로 열기'를 선택하거나, Chrome/Safari 등 기본 브라우저에서 직접 접속해 주세요.

## 문제 해결

### 설치 오류

**증상**: 설치 파일 실행 시 오류가 발생하거나 설치가 진행되지 않음

**해결 방법**:
1. 기존 쌤핀이 설치되어 있다면 먼저 삭제 후 재설치합니다.
2. 관리자 권한으로 실행합니다 (설치 파일 우클릭 → "관리자 권한으로 실행").
3. 백신 프로그램이 차단하는 경우, 일시적으로 예외 처리 후 설치합니다.

### NEIS 연동이 안 됨

**증상**: 시간표나 급식 정보가 표시되지 않음

**해결 방법**:
1. 인터넷 연결을 확인합니다.
2. 설정 → 학교 정보가 올바르게 입력되어 있는지 확인합니다.
3. NEIS API 점검 시간(보통 새벽)에는 일시적으로 연동이 되지 않을 수 있습니다.
4. 학년, 반 정보가 올바른지 확인합니다.

### 화면 깨짐

**증상**: UI가 깨져 보이거나 글자가 겹침

**해결 방법**:
1. 앱을 종료 후 재시작합니다.
2. Windows 디스플레이 배율 설정을 확인합니다 (100% 또는 125% 권장).
3. 앱 최신 버전으로 업데이트합니다.

### 데이터 유실

**증상**: 저장한 데이터가 사라짐

**해결 방법**:
1. `%APPDATA%/ssampin/data/` 폴더에서 JSON 파일이 있는지 확인합니다.
2. 백업 파일이 있다면 설정 → 복원 기능을 사용합니다.
3. Google Drive 동기화를 사용 중이라면 클라우드에서 복원할 수 있습니다.
4. 앱을 삭제해도 데이터는 별도 폴더에 저장되므로 재설치 후 자동으로 복원됩니다.

### 급식 미표시

**증상**: 급식 정보가 "급식 정보 없음"으로 표시됨

**해결 방법**:
1. 인터넷 연결을 확인합니다.
2. 설정에서 학교가 등록되어 있는지 확인합니다.
3. 방학 기간이나 학교 급식이 없는 날에는 표시되지 않습니다.
4. NEIS API 점검 시간에는 일시적으로 표시되지 않을 수 있습니다.
