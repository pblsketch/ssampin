## Design Context

### Users
- **대상**: 한국 중/고등학교 교사 (담임교사 중심)
- **사용 맥락**: 수업 중 시간표 확인, 쉬는 시간에 좌석 배치 변경, 방과 후 학생 기록 정리
- **핵심 Job**: 교사의 일상 업무(시간표, 좌석배치, 일정, 메모, 할일, 학생기록)를 하나의 앱으로 통합 관리
- **기술 수준**: 비개발자. 직관적이고 즉시 이해 가능한 UI 필수
- **기기**: Windows 데스크톱 (Electron), 보조적으로 모바일 PWA

### Brand Personality
- **3단어**: 신뢰감 있는(Trustworthy), 깔끔한(Clean), 따뜻한(Warm)
- **톤**: 전문적이지만 딱딱하지 않음. 교사가 매일 쓰는 도구답게 편안하고 안정감 있어야 함
- **감정 목표**: "이거 하나면 충분하다"는 안도감, 정리된 데스크 같은 뿌듯함
- **이름 유래**: 쌤(선생님) + 핀(중요한 것을 꽂아두다) — 선생님의 대시보드

### Aesthetic Direction
- **비주얼 톤**: 정돈된 다크 대시보드. 데이터 밀도가 높지만 시각적으로 숨쉴 공간이 있음
- **기본 테마**: 다크 블루 (`#0a0e17` 배경) — 장시간 사용해도 눈의 피로가 적음
- **멀티 테마**: 9개 프리셋 (dark, light, pastel, navy, forest, sunset, mono, notion-light, notion-dark) + 커스텀 테마
- **참고 레퍼런스**: Notion (깔끔한 정보 구조), Linear (세련된 다크 UI), 교무수첩 (친숙한 정보 배치)
- **안티 레퍼런스**: 학교 행정 시스템 (복잡하고 구식), 게이머 대시보드 (과도한 네온/글로우)

### Design Principles
1. **정보 우선(Information First)**: 장식보다 데이터. 교사가 3초 안에 필요한 정보를 찾을 수 있어야 함
2. **일관된 테마 적응(Theme Coherence)**: 모든 UI 요소가 `--sp-*` CSS 변수를 통해 테마 전환에 완벽히 적응. 하드코딩 색상 금지
3. **한글 타이포그래피(Korean Typography)**: Noto Sans KR 기본, 10+ 한글 폰트 지원. 한글의 자간/행간 최적화 우선
4. **점진적 복잡성(Progressive Complexity)**: 기본 화면은 단순하게, 고급 기능은 상호작용을 통해 드러남
5. **오프라인 안정성(Offline Reliability)**: 로컬 JSON 저장. 네트워크 없이도 모든 기능 동작

### Design System

#### Color Tokens (`--sp-*` CSS Variables)
| Token | Purpose | Dark Default | Light Default |
|-------|---------|-------------|---------------|
| `--sp-bg` | 최하단 배경 | `#0a0e17` | `#e0e2e6` |
| `--sp-surface` | 사이드바, 서피스 | `#131a2b` | `#d7d9de` |
| `--sp-card` | 카드 배경 (via `--sp-card-base`) | `#1a2332` | `#e6e7eb` |
| `--sp-border` | 테두리 | `#2a3548` | `#b0b5bf` |
| `--sp-accent` | 주 강조색 (파란) | `#3b82f6` | `#2563eb` |
| `--sp-accent-fg` | 강조 배경 위 텍스트 (자동 계산) | `#ffffff` | auto |
| `--sp-highlight` | 보조 강조 (앰버) | `#f59e0b` | `#d97706` |
| `--sp-text` | 기본 텍스트 | `#e2e8f0` | `#0f172a` |
| `--sp-muted` | 보조 텍스트 | `#94a3b8` | `#64748b` |

#### Theme Mechanism
- Tailwind CSS 확장: `bg-sp-card`, `text-sp-text`, `border-sp-border` 등
- `.theme-light` / `.theme-dark` 클래스로 테마 전환
- `useThemeApplier` 훅이 런타임에 CSS 변수 주입
- 과목별 색상 배지: 국어=yellow, 영어=green, 수학=blue, 과학=purple, 사회=orange 등

#### Typography
- 기본 폰트: Noto Sans KR (400 Regular, 700 Bold)
- 10+ 대체 폰트 지원 (Pretendard, IBM Plex Sans KR, SUIT 등)
- 아이콘: Material Symbols Outlined (Google Fonts CDN)
- 모든 UI 텍스트: 한국어

#### Spacing & Layout
- 카드 모서리: `rounded-xl` (12px) 기본, 테마별 조정 가능
- 카드 간격: 16px 기본 (`--sp-card-gap`)
- 4px 그리드 기반 간격 (Tailwind: gap-1, gap-2, gap-3, gap-4)

#### Shadows
- `none` (기본), `sm`, `md`, `lg` 4단계
- 테마별 설정 가능 (`--sp-card-shadow`)
